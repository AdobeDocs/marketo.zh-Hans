# 取消计划的重复批活动运行

<br> 

如果您有不再需要的重复批活动，可以取消将来的运行。

## 取消整个重复

1. 选择活动并转至“ [!UICONTROL **计划**] ”选项卡。

   ![图像1](/help/sky/assets/smart-campaigns/cancel-a-scheduled-recurring-batch-campaign-run/cancel-a-scheduled-recurring-batch-campaign-run-1.png)

1. 单击“ [!UICONTROL **编辑重复**]”。

   ![图像2](/help/sky/assets/smart-campaigns/cancel-a-scheduled-recurring-batch-campaign-run/cancel-a-scheduled-recurring-batch-campaign-run-2.png)

1. 将“循环”设置为“无”，然后单击“ [!UICONTROL **确认**]”。

   ![图3](/help/sky/assets/smart-campaigns/cancel-a-scheduled-recurring-batch-campaign-run/cancel-a-scheduled-recurring-batch-campaign-run-3.png)

>[!CAUTION]
>
>这会取消将来的运行，但如果智能活动正在运行，则无法取消它。

## 取消单个运行

1. 选择活动并转至“ [!UICONTROL **计划**] ”选项卡。

   ![图像4](/help/sky/assets/smart-campaigns/cancel-a-scheduled-recurring-batch-campaign-run/cancel-a-scheduled-recurring-batch-campaign-run-4.png)

1. 单击“ [!UICONTROL **计划运行**]”。

   ![图像5](/help/sky/assets/smart-campaigns/cancel-a-scheduled-recurring-batch-campaign-run/cancel-a-scheduled-recurring-batch-campaign-run-5.png)

1. 查找要取消的单次运行，并 [!UICONTROL **单击最**] 右侧的X。

   ![图6](/help/sky/assets/smart-campaigns/cancel-a-scheduled-recurring-batch-campaign-run/cancel-a-scheduled-recurring-batch-campaign-run-6.png)

>[!NOTE]
>
>单击铅笔图标可编辑该单次运行。
