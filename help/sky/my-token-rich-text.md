# 我的令牌：富文本

<br> 

下面介绍如何设置富文本令牌以用于项目和活动文件夹。

1. 在项目 [!UICONTROL My Tokens] 或文件夹的选项卡中，将富文 **本标记拖** 到画布 [!UICONTROL Local Tokens] 上。

   ![图像1](/help/sky/assets/my-tokens/my-token-rich-text/my-token-rich-text-1.png)

1. 输入新令牌的名称，然后在编辑器中添加所需的任何文本。 单击 [!UICONTROL **保存**]。

   ![图像2](/help/sky/assets/my-tokens/my-token-rich-text/my-token-rich-text-2.png)
