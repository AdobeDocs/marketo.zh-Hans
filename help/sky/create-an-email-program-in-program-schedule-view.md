# 在项目计划项目中创建电子邮件视图

<br> 

您可以直接在项目计划项目中创建新的电子邮件视图。

1. 在树中选择项目，然后转到&#x200B;**[!UICONTROL Overview]**&#x200B;下的&#x200B;**[!UICONTROL Schedule]**&#x200B;选项卡。

   ![图像1](/help/sky/assets/program-schedule-view/create-an-email-program-in-program-schedule-view/create-an-email-program-in-program-schedule-view-1.png)

1. 将鼠标悬停在要添加条目的日期上，然后单击&#x200B;**[!UICONTROL Add]**。

   ![图2](/help/sky/assets/program-schedule-view/create-an-email-program-in-program-schedule-view/create-an-email-program-in-program-schedule-view-2.png)

   >[!NOTE]
   >
   >您还可以按照类似步骤从“日”、“周”或“议程”项目创建电子邮件视图。

1. 输入新智能活动的名称，然后将&#x200B;**[!UICONTROL Type]**&#x200B;更改为[!UICONTROL Email Program]。 如果要在创建条目后保留在日历中，您还可以修改&#x200B;**[!UICONTROL Start Date]**&#x200B;并取消选中&#x200B;**[!UICONTROL Open In Agenda View]**。 单击&#x200B;**[!UICONTROL Save]**。

   ![图3](/help/sky/assets/program-schedule-view/create-an-email-program-in-program-schedule-view/create-an-email-program-in-program-schedule-view-3.png)

1. 您现在可以在日历中找到电子邮件项目。 将鼠标悬停在它上方可查看详细信息……

   ![图4](/help/sky/assets/program-schedule-view/create-an-email-program-in-program-schedule-view/create-an-email-program-in-program-schedule-view-4.png)

...或单击打开编辑器窗口。

![图像5](/help/sky/assets/program-schedule-view/create-an-email-program-in-program-schedule-view/create-an-email-program-in-program-schedule-view-5.png)

>[!NOTE]
>
>您可以在[!DNL Marketo Sky]的项目计划视图内创建电子邮件项目，但实际项目设置将位于[!DNL Marketo Classic]中。
