# 在项目计划项目中创建电子邮件视图

<br> 

您可以直接在项目视图中创建新的电子邮件项目计划。

1. 在树中选择项目，然后转到“概 [!UICONTROL **述**] ”下的 [!UICONTROL **计划选项卡**]。

   ![图像1](/help/sky/assets/program-schedule-view/create-an-email-program-in-program-schedule-view/create-an-email-program-in-program-schedule-view-1.png)

1. 将指针悬停在要添加条目的日期上，然后单击“添 [!UICONTROL **加”**]。

   ![图像2](/help/sky/assets/program-schedule-view/create-an-email-program-in-program-schedule-view/create-an-email-program-in-program-schedule-view-2.png)

   >[!NOTE]
   >
   >您还可以按照类似步骤从“日”、“周”或“议程”项目中创建电子邮件视图。

1. 输入新智能活动的名称，然后将“类 [!UICONTROL **型**] ”更 [!UICONTROL Email Program]改。 如果要在创建条目 [!UICONTROL **后保留在日历中**]，还可以修改 [!UICONTROL **开始日期**] ，并取消选中“在议程中打开”视图。 单击 [!UICONTROL **保存**]。

   ![图3](/help/sky/assets/program-schedule-view/create-an-email-program-in-program-schedule-view/create-an-email-program-in-program-schedule-view-3.png)

1. 您现在可以在日历中找到电子邮件项目。 将鼠标悬停在它上方可获取详细信息……

   ![图像4](/help/sky/assets/program-schedule-view/create-an-email-program-in-program-schedule-view/create-an-email-program-in-program-schedule-view-4.png)

...或单击以打开编辑器窗口。

![图像5](/help/sky/assets/program-schedule-view/create-an-email-program-in-program-schedule-view/create-an-email-program-in-program-schedule-view-5.png)

>[!NOTE]
>
>您可以在中的项目计划项目内创 [!DNL Marketo Sky]建电子邮件视图，但实际项目设置将处于 [!DNL Marketo Classic]中。
