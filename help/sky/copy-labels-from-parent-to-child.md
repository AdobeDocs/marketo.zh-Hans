# 将标签从父项复制到子项

<br> 

要将标签从父资产复制到子资产，请导航到资产[!UICONTROL Overview]选项卡下的[!UICONTROL Details]页面。

在[!UICONTROL Labels]部分下，单击&#x200B;**[!UICONTROL Copy Labels From Parent]**。

![图像1](/help/sky/assets/labels/copy-labels-from-parent-to-child/copy-labels-from-parent-to-child-1.jpg)

复制的标签会自动填充。

![图2](/help/sky/assets/labels/copy-labels-from-parent-to-child/copy-labels-from-parent-to-child-2.jpg)
