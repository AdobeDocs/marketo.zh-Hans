# 在项目计划视图中编辑条目

<br> 

1. 多次-单击要编辑的条目。 这可以从任何日历视图中完成。

   ![图像1](/help/sky/assets/program-schedule-view/edit-an-entry-in-program-schedule-view/edit-an-entry-in-program-schedule-view-1.png)

1. 在弹出窗口中，可编辑条目的不同元素。 如果条目是智能活动或电子邮件项目，请单击条目标题以打开活动/项目概述。

   ![图像2](/help/sky/assets/program-schedule-view/edit-an-entry-in-program-schedule-view/edit-an-entry-in-program-schedule-view-2.png)

>[!NOTE]
>
>基本条目的名称在创建后便无法更改。

>[!NOTE]
>
>已运行的智能活动和电子邮件项目条目无法移至过去。
