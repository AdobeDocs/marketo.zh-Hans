# 项目中的本地资产

<br> 

本地资产是组成您的项目的内容。 资产可自定义，使您能够构建自动化的营销计划。 以下是您可以在中创建的大多数本地资源 [!DNL Marketo Sky]。

![图像1](/help/sky/assets/programs/local-assets-in-a-program/local-assets-in-a-program-1.jpg)

* [智能活动](https://docs.marketo.com/display/MER/Smart+Campaigns)
* [列表](https://docs.marketo.com/display/DOCS/Understanding+Static+Lists)
* [电子邮件项目](https://docs.marketo.com/display/DOCS/Email+Programs)
* [电子邮件](https://docs.marketo.com/display/DOCS/Create+an+Email+for+an+Email+Program)
* [登陆页](https://docs.marketo.com/display/DOCS/Landing+Pages)
* [Forms](https://docs.marketo.com/display/DOCS/Forms)

>[!IMPORTANT]
>
>并非所有客户都有权访问每个可用资产。 请联系您的客户代表以了解更多信息。
