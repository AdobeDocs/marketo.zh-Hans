# 发行说明： 2012年5月 {#release-notes-may}

## 电子邮件性能报表重新设计 {#email-performance-report-redesign}

注意：这将是一个从5月版本开始的分阶段推出

我们使电子邮件性能和促销活动电子邮件性能报表运行得更快。 我们还改进了某些量度的定义，并将“已发送邮件”和“已发送潜在客户”量度整合为单个量度“已发送”。 我们已将“Messages Delivered”和“Lead Delivered”合并为“Delivered”。

## 等待步骤增强功能 {#wait-step-enhancements}

使用新的高级等待属性，您可以在Smart Campaign流量操作中配置等待步骤，以“等待”一周中的特定日期、下一个工作日、特定日期或时间。 这些增强功能可确保您的培养电子邮件在工作时间到达收件箱！

![](assets/image2014-9-23-10-3a14-3a13.png)

图1. 指定要在工作日结束的等待步骤

## 已存档的资产已隐藏 {#archived-assets-hidden}

存档的资产会自动从自动建议、下拉列表和报告中进行筛选，以便更轻松地找到您要查找的内容！

![](assets/image2014-9-23-10-3a14-3a28.png)
