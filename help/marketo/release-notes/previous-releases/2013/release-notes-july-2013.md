# 发行说明：2013年7月 {#release-notes-july}

7月版中包含以下功能，该版本计划于7月26日星期五推出。

## 功能板上已用尽的内容小组件 {#exhausted-content-widget-on-the-dashboard}

提供有关潜在客户何时会耗尽流中的内容的信息。 系统将向您提供有关将要达到耗尽内容的潜在客户数量，或潜在客户已耗尽的时间的信息。

![](assets/image2014-9-22-16-3a30-3a50.png)

## 通信限制 {#communication-limits}

想停止过量发送潜在客户吗？ 现在，自动将频率限制在每个人上很容易。 只需设置每日和每周的通信限制，系统就会完成其余的操作。 适用于Select、Enterprise和Standard客户的附加组件包。

![](assets/image2014-9-22-16-3a31-3a13.png)

## Cobalt用户界面 {#cobalt-user-interface}

在接下来的几个月中，您会注意到我们在应用程序不同部分推出了更多新主题。 将不会移动或删除任何功能。

![](assets/image2014-9-22-16-3a31-3a42.png)

## 程序成员日期列 {#program-member-date-column}

按添加潜在客户的日期查看并排序成员网格。

![](assets/image2014-9-22-16-3a32-3a1.png)

## 对WYSIWYG编辑器中拼写检查的更改 {#changes-to-spell-check-in-wysiwyg-editor}

WYSIWYG编辑器用于拼写检查的服务已停止。 我们从编辑器中删除了拼写检查按钮，直到找到替换项。
