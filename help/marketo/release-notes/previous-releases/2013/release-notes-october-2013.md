# 发行说明：2013年10月 {#release-notes-october}

2013年10月版本中包含以下功能。

## templates.marketo.com {#templates-marketo-com}

[Templates.marketo.com](/help/marketo/product-docs/demand-generation/landing-pages/landing-page-templates/guided-landing-page-template-list.md) 显示电子邮件和登陆页面模板（包括响应式移动电子邮件模板），您可以从Marketo计划库下载这些模板。 我们将每月添加模板，并经常回访！

## developers.marketo.com {#developers-marketo-com}

[Developers.marketo.com](https://developers.marketo.com) 适用于要构建集成到Marketo的开发人员。 您可以引用不同的集成选项，包括Munchkin JavaScript API、SOAP API代码示例、Webhook和电子邮件脚本。 Java SDK也位于 [GitHub](https://github.com/Marketo/SOAP-API-Java-Client).

## 更新了BrightTALK事件适配器 {#updated-brighttalk-event-adapter}

将其他字段从BrightTALK同步到Marketo，包括公司名称、职位、行业和公司规模。

## Android平板电脑事件签入应用程序 {#android-tablet-event-check-in-app}

使用我们的 [基于Android的应用程序中检查](/help/marketo/product-docs/core-marketo-concepts/mobile-apps/event-check-in/check-people-into-your-event-from-your-tablet.md) 可在 [Google Play](https://play.google.com/store/apps/details?id=com.marketo.eventcheckin&amp;hl=en).
