# 发行说明：2014年10月 {#release-notes-october}

查看Marketo版本以了解功能可用性。 文档将在发布时提供。

## 营销日历中的项目聚焦 {#program-focus-in-marketing-calendar}

[创建和编辑条目](/help/marketo/product-docs/core-marketo-concepts/marketing-calendar/understanding-the-calendar/understand-enable-program-focus.md) 直接从营销日历。

![](assets/image2014-10-20-11-3a48-3a51.png)

## 新的ReST API调用 {#new-rest-api-calls}

使用API向潜在客户拉取新活动或更改：

* 获取潜在客户更改
* 获取潜在客户活动
* 获取活动类型
* 获取分页令牌

完整详细信息将在发布后提供，网址为 [developers.marketo.com](https://developers.marketo.com/documentation/rest/).

## MSI — 发送Microsoft Dynamics的Marketo电子邮件 {#msi-send-marketo-email-for-microsoft-dynamics}

[发送和跟踪销售电子邮件](/help/marketo/product-docs/marketo-sales-insight/msi-for-microsoft-dynamics/setting-up-and-using/send-a-marketo-sales-email-from-microsoft-dynamics.md) Microsoft Dynamics的潜在客户和联系人。

![](assets/image2014-10-20-11-3a49-3a25.png)

## MSI — 添加到Marketo Campaigns for Microsoft Dynamics {#msi-add-to-marketo-campaigns-for-microsoft-dynamics}

[将潜在客户和联系人添加到Marketo智能营销活动](/help/marketo/product-docs/marketo-sales-insight/msi-for-microsoft-dynamics/setting-up-and-using/add-a-lead-contact-to-a-marketo-campaign-from-microsoft-dynamics.md) 直接从Microsoft Dynamics中。 营销人员可以选择向销售人员提供Marketo促销活动。

## Microsoft Dynamics Sync的自定义实体支持 {#custom-entity-support-for-microsoft-dynamics-sync}

[使用自定义对象数据](/help/marketo/product-docs/crm-sync/microsoft-dynamics-sync/microsoft-dynamics-sync-details/enable-sync-for-a-custom-entity.md) 从Microsoft Dynamics中过滤和触发智能列表、智能营销活动、项目……

## Microsoft Dynamics Sync的股东支持 {#shareholder-support-for-microsoft-dynamics-sync}

从Dynamics同步商机股东数据。 另外，使用“主要客户”字段连接到客户的商机，以及使用“主要联系人”同步连接到联系人的商机，也受支持。

## RTP — 功能板增强功能 {#rtp-dashboard-enhancements}

功能板现在已得到增强，可以包含更多一览表数据：

* 组织访问次数总计
* 绩效最佳的前5个行业
* 参与访客总数

## RTP — 新的营销活动移动模板 {#rtp-new-mobile-templates-for-campaigns}

快速轻松 [创建移动营销活动](/help/marketo/product-docs/web-personalization/using-templates/using-templates-to-create-web-campaigns.md) 这些新的模板。

## RTP — 用户上下文API {#rtp-user-context-api}

使用跟踪访客过去访问历史记录的新调用。 根据访客的以下内容个性化营销活动：

* 已查看的过去页面
* 感兴趣的产品
* 他们所看到的RTP促销活动

访问 [developers.marketo.com](https://developers.marketo.com/documentation/websites/rtp-js-api/) 以了解完整的详细信息。
