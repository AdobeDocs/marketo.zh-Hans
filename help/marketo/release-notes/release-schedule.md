# 发行计划 {#release-schedule}

**版本**:Marketo使用季度主要发行计划。一旦发布，每个发行说明都将链接在下表中。

**发行日期**:推出通常从发行周的星期三开始，大部分升级工作在周五晚上进行。系统会发送一封包含特定日期和发行详细信息的电子邮件。 列出的日期是星期五的更新。

**进度**:

* 按计划 — 我们计划在指定日期发布季度功能和增强功能
* 正在进行 — 我们正在开发并记录内容，以便在指定日期发布
* 已发运 — 版本已完成

_发行日期可能会有变_
