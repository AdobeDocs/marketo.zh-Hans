# 发布计划 {#release-schedule}

**版本**：每个版本的年/月。 发行说明一经发布，即会链接到下表。

**发行日期**：推出通常从发布周的星期二开始，大部分升级在星期五晚上进行。 将发送一封包含特定日期和发行详细信息的电子邮件。 列出的日期用于星期五更新。

**状态**:

* 按计划 — 我们计划在指定日期发布功能和增强功能
* 进行中 — 我们正在推出该版本的功能
* 已发货 — 版本已完成

_发行日期可能会有变动_
