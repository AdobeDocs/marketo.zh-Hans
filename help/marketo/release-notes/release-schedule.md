# 发布计划 {#release-schedule}

**版本**：每个版本的月份/年份。 发行说明一经发布，即会链接到下表。

**发行日期**：转出通常从发布周的周二开始，更多升级在周五晚上进行。 列出的日期用于星期五更新。

**状态**:

* 按计划 — 我们计划在指定日期发布功能和增强功能
* 进行中 — 我们正在推出此版本的功能
* 已发货 — 版本已完成

_发行日期/功能可能会发生更改_
