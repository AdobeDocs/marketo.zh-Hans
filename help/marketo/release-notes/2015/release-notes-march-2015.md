# 发行说明：2015年3月{#release-notes-march}

2015年3月版包含以下功能。 请查看您的Marketo Edition以了解功能可用性。 发布后，请务必返回以找到每个功能的详细文章的链接。

## 日历HD {#calendar-hd}

[使用日历的新演示](http://docs.marketo.com/display/docs/calendar+hd) 模式显示团队的营销活动。这些对于电视或办公室周围的巨型显示器都很好！ 根据智能列表或自定义指标设置和显示目标。

>[!NOTE]
>
>此功能不适用于Spark和Standard版本。

![](assets/image2015-3-23-11-3a39-3a15.png)

## Google Adwords集成{#google-adwords-integration}

将您的[Google AdWords帐户链接到Marketo](../../product-docs/administration/additional-integrations/add-google-adwords-as-a-launchpoint-service.md)，以自动将脱机转换数据从Marketo上传到Google AdWords。 然后，从AdWords UI中，您将能够轻松地看到哪些点击产生了合格的潜在客户、机会和新客户（或要跟踪的任何收入阶段）。

![](assets/image2015-3-23-11-3a50-3a55.png)

## 重新设计收入浏览器{#revenue-explorer-redesign}

[Revenue ](http://docs.marketo.com/display/docs/revenue+explorer) Explorer具有全新的外观和风格，以及新的Sunburst图表类型！我们将在4月的头两周推出。
