# 发行说明：2015年3月 {#release-notes-march}

2015年3月版包含以下功能。 请查看您的Marketo Edition以了解功能可用性。 发布后，请务必返回以找到每个功能的详细文章的链接。

## 日历高清 {#calendar-hd}

[使用日历的新演示模式](http://docs.marketo.com/display/docs/calendar+hd) ，显示您的团队的营销活动。 这些对于电视或办公室周围的巨型显示器都很好！ 根据智能列表或自定义指标设置和显示目标。

>[!NOTE]
>
>此功能不适用于Spark和Standard版本。

![](assets/image2015-3-23-11-3a39-3a15.png)

## Google Adwords集成 {#google-adwords-integration}

将您的 [Google AdWords帐户关联到Marketo](../../product-docs/administration/additional-integrations/add-google-adwords-as-a-launchpoint-service.md) ，以自动将线下转换数据从Market上传到Google AdWords。 然后，从AdWords UI中，您将能够轻松地看到哪些点击产生了合格的潜在客户、机会和新客户（或要跟踪的任何收入阶段）。

![](assets/image2015-3-23-11-3a50-3a55.png)

## 收入浏览器重新设计 {#revenue-explorer-redesign}

[Revenue Explorer](http://docs.marketo.com/display/docs/revenue+explorer) （收入浏览器）具有全新的外观和风格，以及全新的Sunburst图表类型！ 我们将在4月的头两周推出。

## 新的资产REST API {#new-asset-rest-apis}

[新的资产REST API](http://developers.marketo.com/)

我们现在支持通过API创建和编辑电子邮件、模板、我的令牌、文 [件和代码片段](http://developers.marketo.com/documentation/asset-api)!

## Microsoft Dynamics 2015内部部署 {#microsoft-dynamics-on-premise}

现在可通过应用程序访问 [最新安装程序](../../product-docs/crm-sync/microsoft-dynamics-sync/sync-setup/download-the-marketo-lead-management-solution/upgrade-the-marketo-solution-for-microsoft-dynamics.md)。

![](assets/image2015-3-23-11-3a47-3a16.png)

## RTP —— 利用潜在客户数据实现个性化的Web参与 {#rtp-personalized-web-engagement-with-lead-data}

利用您 [在营销](../../product-docs/web-personalization/using-web-segments/manage-person-data.md)人员中拥有的潜在客户数据字段，创建实时细分和个性化内容活动。 管理RTP中的潜在客户数据字段并添加／删除相关的潜在客户字段。
