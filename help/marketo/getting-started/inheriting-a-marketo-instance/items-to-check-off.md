# 要结帐的项目 {#items-to-check-off}

确保使用正确的访问和信息类型正确设置您。 执行这些简单步骤，以便您能够立即查看待办事项列表中的某些内容！

* 观看简短的 [**Marketo做什么**](https://pages2.marketo.com/demoFull.html){target=&quot;_blank&quot;}。

* **设置配置文件** 在 [**营销国家**](https://nation.marketo.com/){target=&quot;_blank&quot;}社区（需要用户名和密码）。

* 了解如何与 [**营销国家**](https://nation.marketo.com/t5/About-Community/ct-p/about-community){target=&quot;_blank&quot;}社区，以获取您的问题的答案。

* 单击 **您的位置** 在您的Marketo国家/地区社区用户档案中，查看您公司的帐户页面。

* 了解如何 [**提交支持案例**](https://nation.marketo.com/t5/Knowledgebase/Submitting-a-Support-Case-to-Marketo-Support/ta-p/252201){target=&quot;_blank&quot;}(并确认您 [**设置为授权支持联系人**](https://nation.marketo.com/t5/Knowledgebase/Managing-Authorized-Support-Contacts/ta-p/254341){target=&quot;_blank&quot;})。

* [**管理用户**](/help/marketo/product-docs/administration/users-and-roles/managing-marketo-users.md){target=&quot;_blank&quot;}并删除那些离开了您公司的用户。
