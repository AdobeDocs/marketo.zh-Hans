# 继承文档1 {#inherit-doc-1}

对继承的实例进行审计可能看起来像是

您是否从其他管理员继承了现有Marketo Engage实例？ 如果是这样的话，这篇文章是为您准备的。

>[!TIP]
>
>如果您是新Marketo Engage用户并且不熟悉许多术语，请查看 [Marketo术语表](/help/marketo/getting-started/marketo-glossary.md){target="_blank"}.
