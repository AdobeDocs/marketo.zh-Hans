# 新实例最佳实践：营销活动核对清单 {#new-instance-best-practices-marketing-activities-checklist}

营销活动包含构成自动营销计划的所有资源和内容。 在设置新的Marketo Engage实例时，一个简洁明了的机构可确保所有用户都能轻松找到和管理各种程序。

请记得下载核对清单 [链接] 并跟踪您的进度。
