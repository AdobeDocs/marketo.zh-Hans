# 查看组织服务URL {#view-the-organization-service-url}

Marketo Engage需要组织服务URL才能与MD实例同步。 下面是如何在Dynamics中找到它。

1. 登录到[!DNL Dynamics]。 单击“设置”图标并选择&#x200B;**[!UICONTROL 高级设置]**。

   ![](assets/one.png)

1. 单击&#x200B;**[!UICONTROL 设置]**&#x200B;并选择&#x200B;**[!UICONTROL 自定义项]**。

   ![](assets/two.png)

1. 单击&#x200B;**[!UICONTROL 开发人员资源]**。

   ![](assets/three.png)

1. 可在&#x200B;**[!UICONTROL 服务终结点]**&#x200B;下找到组织服务URL。

   ![](assets/four.png)

1. 将该URL复制并粘贴到Marketo中，然后尽情体验其余的同步过程。
