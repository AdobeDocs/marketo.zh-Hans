# Microsoft Dynamics同步：Opportunity Sync {#microsoft-dynamics-sync-opportunity-sync}

Marketo到Dynamics的同步功能非常强大。 以下是机会同步的所有详细信息：

## 如何使两个系统之间的机会详细信息保持同步？{#how-are-opportunity-details-kept-in-sync-between-the-two-systems}

机会同步是一种方式 — Dynamics与Marketo。 如果您在Dynamics中更改了某个机会，您的更新将反映在Marketo中。

## 我是否可以使用Marketo在Dynamics中创建机会？{#can-i-create-an-opportunity-in-dynamics-using-marketo}

否，您必须在Dynamics中创建此机会，它将自动同步到Marketo。

## 哪些字段将同步到Marketo?{#what-fields-will-sync-to-marketo}

您可以在设置过程中[选择要同步的字段](/help/marketo/product-docs/crm-sync/microsoft-dynamics-sync/sync-setup/microsoft-dynamics-365/step-3-of-3-connect.md#select-fields-to-sync)。
