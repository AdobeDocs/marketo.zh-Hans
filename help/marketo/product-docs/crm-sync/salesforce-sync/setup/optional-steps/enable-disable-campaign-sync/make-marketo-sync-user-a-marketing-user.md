# 使Marketo Sync用户成为营销用户{#make-marketo-sync-user-a-marketing-user}

[Marketo sync用户](/help/marketo/product-docs/crm-sync/salesforce-sync/setup/enterprise-unlimited-edition/step-2-of-3-create-a-salesforce-user-for-marketo-enterprise-unlimited.md)需要为Salesforce活动同步的Marketing用户才能正常工作。 以下是如何在Salesforce中使用户成为营销用户。

>[!NOTE]
>
>**需要管理权限**

1. 登录到Salesforce。 在左搜索栏中搜索用户，然后单击“管理用户”下的“**用户**”。****

   ![](assets/image2015-7-8-14-3a25-3a49.png)

1. 查找同步用户并单击其姓名。

   ![](assets/image2015-7-8-14-3a27-3a32.png)

1. 单击&#x200B;**编辑**。

   ![](assets/image2015-7-8-14-3a29-3a7.png)

1. 选中&#x200B;**Marketing User**&#x200B;复选框，然后单击&#x200B;**Save**。

   ![](assets/image2015-7-8-14-3a30-3a16.png)

   太棒了！ 此Marketo同步用户现在是营销用户。
