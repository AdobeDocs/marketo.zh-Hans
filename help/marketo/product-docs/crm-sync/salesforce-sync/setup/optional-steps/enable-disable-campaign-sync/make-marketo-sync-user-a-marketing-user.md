# 将Marketo同步用户设为营销用户 {#make-marketo-sync-user-a-marketing-user}

的 [Marketo同步用户](/help/marketo/product-docs/crm-sync/salesforce-sync/setup/enterprise-unlimited-edition/step-2-of-3-create-a-salesforce-user-for-marketo-enterprise-unlimited.md) 需要是Salesforce促销活动同步的营销用户才能正常工作。 以下是如何在Salesforce中将用户设为营销用户。

>[!NOTE]
>
>**需要管理员权限**

1. 登录到Salesforce。 在左侧搜索栏中搜索用户，然后单击 **用户** 在 **管理用户**.

   ![](assets/image2015-7-8-14-3a25-3a49.png)

1. 查找同步用户并单击其名称。

   ![](assets/image2015-7-8-14-3a27-3a32.png)

1. 单击 **编辑**.

   ![](assets/image2015-7-8-14-3a29-3a7.png)

1. 检查 **营销用户** 复选框，单击 **保存**.

   ![](assets/image2015-7-8-14-3a30-3a16.png)

   太棒了！ 此Marketo同步用户现在是营销用户。
