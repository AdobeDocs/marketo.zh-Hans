# 启用/禁用营销活动同步 {#enable-disable-campaign-sync}

>[!NOTE]
>
>**需要管理员权限**

此选项允许Marketo将其项目成员资格和状态与Salesforce促销活动同步，反之亦然。

1. 转到 **管理员**，单击 **Salesforce**.

   ![](assets/image2014-12-9-13-3a36-3a49.png)

1. 单击 **编辑同步设置**.

   ![](assets/image2014-12-9-13-3a37-3a0.png)

1. 检查 **启用Salesforce促销活动同步** 单击 **保存**.

   ![](assets/image2014-12-9-13-3a37-3a8.png)

给你。 只需给同步一段时间，即可从Salesforce提取数据，您就可以开始使用。
