# 附加管理员 {#additional-admins}

要更改或删除团队成员，请转到[!UICONTROL Settings]页面，然后单击左侧导航面板中的&#x200B;**[!UICONTROL Team Management]**。
作为帐户的管理员，您可以在此处管理团队中的人员。
