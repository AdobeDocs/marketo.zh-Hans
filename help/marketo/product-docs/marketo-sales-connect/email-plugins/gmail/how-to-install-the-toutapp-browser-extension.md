# 如何安装ToutApp Browser Extension {#how-to-install-the-toutapp-browser-extension}

1. 单击 **Web应用** 程序 [上的“设置”](http://toutapp.com/next#settings)。
1. 在“我的帐户”下，单击“电 **子邮件设置**”。
1. 单击 **插件**。
1. 在“Gmail和Google应用程序”下，单击“ **安装**”。
