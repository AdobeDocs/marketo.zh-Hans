# 如何安装ToutApp浏览器扩展 {#how-to-install-the-toutapp-browser-extension}

1. 单击 **设置** 在 [Web应用程序](https://toutapp.com/next#settings).

1. 在“My Account（我的帐户）”下，单击 **电子邮件设置**.

1. 单击 **插件**.

1. 在Gmail和Google Apps下，单击 **安装**.
