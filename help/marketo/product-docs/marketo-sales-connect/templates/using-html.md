# 使用 HTML {#using-html}

1. 使用在HTML中创建电子邮件的工具（例如Marketo的电子邮件编辑器），从电子邮件中复制源代码。

1. 单击模板编辑器中的&#x200B;**Source**&#x200B;按钮。

1. 将源代码粘贴到此处。

1. 单击 **[!UICONTROL Save]**。

>[!NOTE]
>
>如果您看到错误“错误 — 要删除样式/java/html标记”，则意味着您有一些我们不支持样式。 您应该在Source代码中搜索word样式，并删除从`<style>`到`</style>`的所有内容。
