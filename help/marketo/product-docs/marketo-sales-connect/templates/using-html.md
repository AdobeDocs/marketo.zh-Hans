# 使用HTML {#using-html}

1. 在用于以HTML格式创建电子邮件的工具（例如Marketo的电子邮件编辑器）中复制电子邮件中的源代码
1. 单击模 **板编辑器** 中的“源”按钮。
1. 将源代码粘贴到此处。
1. 单击 **保存**。

>[!NOTE]
>
>如果您看到错误“错误——删除style/java/html标记”，则表示您有一些我们不支持的样式。 您应该搜索单词style的源代码并删除从头到尾的所有 `<style>` 内容 `</style>`。
