# 编辑模板类别 {#edit-a-template-category}

我们目前无法删除整个类别或批量将模板移动到新类别。

您可以执行的操作是，将类别中的每个模板编辑为现有的类别名称或全新的名称。 这会将所有现有模板移动到其他文件夹中。 一旦文件夹为空，它就会消失。

我们正在努力增加大规模实现此目标的能力。
