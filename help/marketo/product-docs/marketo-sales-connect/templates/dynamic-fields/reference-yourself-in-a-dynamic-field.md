# 在动态字段中引用您自己 {#reference-yourself-in-a-dynamic-field}

动态字段是我们合并邮件的形式。

我们允许您使用预定义的属性(例如 `{{my_name}}` 或 `{{my_title}}`. 这些字段允许您在电子邮件模板中快速引用自己。

要将自引用动态字段添加到模板，只需从中选择要使用的字段 **MSE动态字段** 按钮。
