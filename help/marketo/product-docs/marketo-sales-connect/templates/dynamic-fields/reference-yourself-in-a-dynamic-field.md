# 在动态字段中引用您自己 {#reference-yourself-in-a-dynamic-field}

动态字段是我们邮件合并的形式。

我们允许您使用预定义的属性（如或）来个性化您 `{{my_name}}` 的电子邮件 `{{my_title}}`模板。 这些字段允许您快速在电子邮件模板中引用自己。

要向模板添加自引用动态字段，只需从模板编辑器的“MSE动态字段”按 **钮中选择要使用的** 字段。
