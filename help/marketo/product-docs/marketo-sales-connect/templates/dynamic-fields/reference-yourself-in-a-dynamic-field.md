# 在动态字段中引用您自己 {#reference-yourself-in-a-dynamic-field}

动态字段是我们的邮件合并形式。

我们允许您使用诸如之类的预定义属性来个性化电子邮件模板 `{{my_name}}` 或 `{{my_title}}`. 利用这些字段，可在电子邮件模板中快速引用自己。

要向模板添加自引用动态字段，只需从 **MSE动态字段** 按钮。
