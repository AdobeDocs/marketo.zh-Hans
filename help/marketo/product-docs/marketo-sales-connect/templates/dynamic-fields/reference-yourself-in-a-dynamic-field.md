# 在动态字段{#reference-yourself-in-a-dynamic-field}中引用您自己

动态字段是我们邮件合并的形式。

我们允许您使用预定义的属性（如`{{my_name}}`或`{{my_title}}`）个性化您的电子邮件模板。 这些字段允许您快速在电子邮件模板中引用自己。

要向模板添加自引用动态字段，只需从模板编辑器的&#x200B;**MSE动态字段**&#x200B;按钮中选择要使用的字段。
