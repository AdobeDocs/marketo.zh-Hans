# 自定义模板视图 {#customize-template-view}

从 **视图** 从下拉列表中可以看到：所有模板、您的模板、共享模板或未使用的模板（过去90天内未使用的模板）。

此外，您可以通过单击右侧的设置图标，在模板视图中添加/删除列 **视图。**
