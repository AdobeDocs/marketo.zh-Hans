# 自定义模板视图 {#customize-template-view}

从 **查看** 下拉列表中，您可以选择查看：所有模板、模板、共享模板或未使用的模板（过去90天内未使用的模板）。

此外，您还可以通过单击模板右侧的设置图标，在模板视图中添加/删除列 **查看。**
