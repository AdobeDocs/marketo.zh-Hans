# 自定义模板视图{#customize-template-view}

从&#x200B;**视图**&#x200B;下拉列表中，您可以选择查看：所有模板、您的模板、共享模板或未使用的模板（过去90天内未使用的模板）。

此外，您还可以单击&#x200B;**视图右侧的设置图标，在模板视图中添加/删除列。**
