# 从Web 应用程序{#error-when-sending-from-the-web-application}发送时出错

通常，当从[web应用程序](https://toutapp.com/login)生成错误时，这是由于电子邮件或动态字段的格式存在问题。 确保在合成窗口中正确填充所有内容，然后重试发送。
