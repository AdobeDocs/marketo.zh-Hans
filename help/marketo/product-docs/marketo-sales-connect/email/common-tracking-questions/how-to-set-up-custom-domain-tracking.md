# 如何设置自定义域跟踪 {#how-to-set-up-custom-domain-tracking}

我们为您提供为链接配置自定义域跟踪的选项，因此当有人将鼠标悬停在链接上时，它将显示为“go.yourcompany.com”而不是“go.toutapp.com”。

要设置此项，您的IT管理员需要遵循以下路径中列出的步骤： [设置页面](https://toutapp.com/custom_tracking_domain).

完成此设置后，我们将允许列表您在电子邮件中的任何链接显示为“go.yourcompany.com”。
