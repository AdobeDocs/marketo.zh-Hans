# 如何设置自定义域跟踪 {#how-to-set-up-custom-domain-tracking}

我们为您提供了为链接配置自定义域跟踪的选项，因此当某人悬停某个链接时，该链接将改为“go.yourcompany.com”，而不是“go.toutapp.com”。

要进行此设置，您的IT管理员需要按照 [设置页面](https://toutapp.com/custom_tracking_domain).

设置完此设置后，我们将允许列表您电子邮件中的任何链接，显示为“go.yourcompany.com”。
