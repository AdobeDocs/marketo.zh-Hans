# 如何设置自定义域跟踪{#how-to-set-up-custom-domain-tracking}

我们为您提供了为链接配置自定义域跟踪的选项，因此当有人悬停在链接上时，该链接将读为“go.yourcompany.com”，而非“go.toutapp.com”。

要进行设置，您的IT管理员需要按照我们的[设置页面](https://toutapp.com/custom_tracking_domain)中列出的步骤操作。

设置完此设置后，我们将允许列表您电子邮件中的任何链接，显示为“go.yourcompany.com”。
