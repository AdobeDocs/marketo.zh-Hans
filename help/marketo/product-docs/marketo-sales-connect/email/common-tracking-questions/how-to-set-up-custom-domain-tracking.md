# 如何设置自定义域跟踪 {#how-to-set-up-custom-domain-tracking}

我们为您提供选项来配置链接的自定义域跟踪，因此当有人将鼠标悬停在链接上时，它将显示为“go.yourcompany.com”而不是“go.toutapp.com”。

若要进行此设置，您的IT管理员需要执行我们的[设置页面](https://toutapp.com/custom_tracking_domain)中列出的步骤。

完成此设置后，我们将允许列表您在电子邮件中拥有的任何链接，使其显示为“go.yourcompany.com”。
