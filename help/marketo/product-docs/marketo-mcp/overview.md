# MCP概述文档 {#overview}

ATHER的文本：

Marketo Engage的公共MCP服务器是一项可云访问的服务，它将Marketo Engage公共REST API的所有功能公开为人工智能代理提供结构化、可发现的工具。 基于模型上下文协议(MCP)，它将Marketo的营销自动化功能转换为标准化的架构化界面，以便AI系统安全而智能地使用。

MCP服务器不是手动将Marketo端点集成到每个应用程序中，而是作为单个AInative网关，允许MCP兼容代理发现可用的操作、了解所需的参数并通过一致的协议执行操作。

有许多可用工具包含公共REST API和内部API，它们像在UI中一样执行Marketo核心功能，但通过API调用。 随着工具的开发，将不断添加更多工具。

MCP服务器作为自助服务提供，并且可以通过构建端点和验证连接来通过Adobe IO框架进行设置。 有关更多信息和设置详细信息，请访问Adobe Developer网站（在此处添加链接）。

显示为SOAP API的Campaign工具已转换为REST API并已添加。

配置完毕后，要查看可用工具列表，请在此处按照这些说明（超链接）操作。

注意：不可用的工具有“停用营销活动”和“删除静态列表”（待定）。
