# 创建指定帐户 {#create-a-named-account}

请按照以下步骤手动创建指定帐户。

1. 在指定帐户中，单击 **新建** 下拉框，然后选 **择创建指定帐户**。

   ![](assets/two-1.png)

1. 填写所需字段，然后单击“ **创建**”。

   ![](assets/three-1.png)

   >[!TIP]
   >
   >直接单击指定帐户以查看其仪表板。
