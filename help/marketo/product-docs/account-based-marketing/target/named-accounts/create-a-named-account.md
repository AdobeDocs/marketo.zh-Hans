# 创建指定帐户{#create-a-named-account}

请按照以下步骤手动创建指定帐户。

1. 在“指定帐户”中，单击&#x200B;**New**&#x200B;下拉框，然后选择&#x200B;**创建指定帐户**。

   ![](assets/two-1.png)

1. 填写所需字段，然后单击&#x200B;**创建**。

   ![](assets/three-1.png)

   >[!TIP]
   >
   >直接单击指定帐户以查看其仪表板。
