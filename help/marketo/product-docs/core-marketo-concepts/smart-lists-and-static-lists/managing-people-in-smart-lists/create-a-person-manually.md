# 手动创建人员 {#create-a-person-manually}

让个人进入Marketo的方法有很多。 如果需要手动创建一个，请参阅以下操作说明。

>[!CAUTION]
>
>Marketo会 **非** 支持包含表情符号的电子邮件地址。

1. 转到 **数据库**.

   ![](assets/db-1.png)

1. 下 **新**，单击 **新人员**.

   ![](assets/two-2.png)

1. 输入人员信息，然后单击 **创建**.

   ![](assets/three-2.png)

这是在Marketo中添加单个人员的快速轻松的方法！
