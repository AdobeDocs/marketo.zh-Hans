# 创建静态列表 {#create-a-static-list}

静态列表是数据库中已经存在的一组人员。 以下是如何创建一个。

1. 转到&#x200B;**[!UICONTROL Database]**。

   ![](assets/db.png)

1. 点击 **[!UICONTROL New]** 下拉菜单，并选择 **[!UICONTROL New List]**。

   ![](assets/two.png)

1. 选择目标文件夹，为新列表命名，然后单击&#x200B;**[!UICONTROL Create]**。

   ![](assets/three.png)

   您现在有一个空列表可供填写。 了解如何在[此处](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/static-lists/understanding-static-lists.md#ways-to-add-remove-people-from-a-list){target="_blank"}添加用户。

   >[!NOTE]
   >
   >您可以根据自己的喜好将某人添加到列表中，添加次数不限，但只显示一次。 人员将保留在列表中，直到您将其移除。
