# 克隆列表或智能列表 {#clone-a-list-or-smart-list}

不要从头开始创建智能列表，请克隆类似的智能列表并进行更改以节省一些时间。 操作方法如下：

1. 前往 **[!UICONTROL Marketing Activities]**。

   ![](assets/clone-a-list-or-smart-list-1.png)

1. 选择要克隆的智能列表。 在&#x200B;**[!UICONTROL List Actions]**&#x200B;下，单击&#x200B;**[!UICONTROL Clone Smart List]**。

   ![](assets/clone-a-list-or-smart-list-2.png)

1. 输入&#x200B;**[!UICONTROL Name]**&#x200B;并单击&#x200B;**[!UICONTROL Clone]**。

   ![](assets/clone-a-list-or-smart-list-3.png)

做得好！ 您也可以按相同方式克隆常规列表。
