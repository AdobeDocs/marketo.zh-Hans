# 在智能列表{#use-the-data-value-changed-filter-in-a-smart-list}中使用数据值更改过滤器

值在不断变化，并在Marketo数据库中进行更新。 有时，您希望寻找的不是特定的价值，而是价值发生变化的人。 下面介绍如何操作。

>[!PREREQUISITES]
>
>[创建智能列表](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/creating-a-smart-list/create-a-smart-list.md)

1. 转至&#x200B;**营销活动**&#x200B;区域。

   ![](assets/ma.png)

1. 选择智能列表并单击&#x200B;**智能列表**&#x200B;选项卡。

   ![](assets/two-1.png)

1. 找到&#x200B;**数据值已更改**&#x200B;滤镜，并将其拖动到画布上。

   ![](assets/three-1.png)

1. 将&#x200B;**Attribute**&#x200B;设置为要查找数据值更改的字段。

   ![](assets/four.png)

此过滤器将查找对其状态有更改（任何更改）的所有人员。 如果想更漂亮，请尝试[向智能列表滤镜添加约束](/help/marketo/product-docs/core-marketo-concepts/smart-lists-and-static-lists/using-smart-lists/add-a-constraint-to-a-smart-list-filter.md)。
