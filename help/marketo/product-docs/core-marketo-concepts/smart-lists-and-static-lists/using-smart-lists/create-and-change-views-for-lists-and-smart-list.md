# 为列表和智能列表{#create-and-change-views-for-lists-and-smart-list}创建和更改视图

智能列表显示一组默认列。 您是否知道可以编辑心目中的列？ 下面介绍如何操作。

## 创建视图{#create-a-view}

1. 转到列表或智能列表，单击&#x200B;**People**&#x200B;选项卡，然后单击&#x200B;**视图**&#x200B;下拉菜单下的&#x200B;**创建** **视图**。

   ![](assets/smartlist-createview.png)

1. 命名视图并查找要添加或删除的列。

   ![](assets/image2014-9-12-11-3a23-3a53.png)

1. 添加/删除所需列，然后单击&#x200B;**创建**。

   ![](assets/image2014-9-12-11-3a24-3a7.png)

>[!TIP]
>
>要快速搜索，请使用&#x200B;**查找**&#x200B;框。

## 交换机视图{#switch-views}

1. 单击&#x200B;**视图**&#x200B;下拉框，然后选择所需的视图。 根据需要在视图之间来回切换。

   ![](assets/smartlist-customviewchoose.png)

>[!NOTE]
>
> 除默认视图外，您必须先创建第二个视图，才能切换视图。

## 编辑视图{#edit-a-view}

1. 确保在&#x200B;**视图**&#x200B;下拉列表中选择了要编辑的视图。

   ![](assets/smartlist-customviewchoose.png)

1. 单击&#x200B;**编辑**&#x200B;选项。

   ![](assets/smartlist-editcustomview.png)

1. 进行所需的更改，然后单击&#x200B;**保存**。

   ![](assets/image2014-9-12-11-3a27-3a19.png)

## 删除视图{#delete-a-view}

1. 从&#x200B;**视图**&#x200B;下拉菜单中选择要删除的视图的&#x200B;**编辑**&#x200B;选项。

   ![](assets/smartlist-editcustomview.png)

1. 单击&#x200B;**删除**。

   ![](assets/image2014-9-12-11-3a27-3a39.png)

只有您能够在“视图”下拉菜单中看到您创建的自定义视图，因此请告诉您的朋友，他们需要自己创建！
