# 刷新列表或智能列表 {#refresh-a-list-or-smart-list}

如果您运行了智能列表并且已经过了几分钟，则结果现在可能不同 — 请刷新以找出答案。

## 刷新结果 {#refresh-results}

1. 要更新智能列表的&#x200B;**[!UICONTROL 人员]**&#x200B;选项卡中的数据，请单击刷新图标。

   ![](assets/refreshbutton.png)

1. 智能列表会重新运行并显示一个最新的结果集。

   ![](assets/loadingrefresh.png)

>[!TIP]
>
>有时当您运行智能列表并稍后返回时，您可能会在右下角的人数前面看到“关于”一词。 这表示该数字是近似值 — 单击计数本身可刷新它并获取更新的准确计数。
