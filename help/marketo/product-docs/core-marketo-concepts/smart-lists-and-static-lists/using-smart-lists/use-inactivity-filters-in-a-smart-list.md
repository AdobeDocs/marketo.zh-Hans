# 在智能列表中使用非活动筛选器 {#use-inactivity-filters-in-a-smart-list}

您知道吗，您可以在智能列表中查找 *没有* 做点什么吗？ 查看这些非活动筛选器。

1. 转到 **营销活动**.

   ![](assets/login-marketing-activities-3.png)

1. 选择要编辑的智能列表，然后单击 **智能列表** 选项卡。

   ![](assets/smartlist-choose.png)

1. 查找所选的非活动筛选器并将其拖到画布中。 例如，让我们查找尚未访问您任何页面的人员。

   ![](assets/draginactivityfilter.png)

   >[!TIP]
   >
   >中提供了许多过滤器 **非活动筛选器** 文件夹。 搜索 **非** 去看看他们。

1. 选择 **为任意** 运算符。 这会查找过去30天内未访问任何页面的所有用户。

   ![](assets/mysmartlist-people.jpg)
