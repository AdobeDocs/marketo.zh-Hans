# 在智能列表中使用非活动筛选器 {#use-inactivity-filters-in-a-smart-list}

您是否知道可以查找智能列表中的人员 _没有_ 做点什么吗？ 查看这些非活动筛选器。

1. 转到 **[!UICONTROL 营销活动]**.

   ![](assets/login-marketing-activities-3.png)

1. 选择要编辑的智能列表，然后单击 **[!UICONTROL 智能列表]** 选项卡。

   ![](assets/smartlist-choose.png)

1. 查找您选择的非活动筛选器并将其拖到画布中。 例如，我们查找尚未访问过您任何页面的人员。

   ![](assets/draginactivityfilter.png)

   >[!TIP]
   >
   >中提供了许多过滤器 **[!UICONTROL 非活动筛选器]** 文件夹。 搜索“Not”并查看它们。

1. 选择 **[!UICONTROL 为任意]** 运算符。 这会查找过去30天内未访问任何页面的所有人员。

   ![](assets/mysmartlist-people.jpg)
