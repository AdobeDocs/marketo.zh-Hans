# 跨工作区引用列表或智能列表 {#reference-a-list-or-smart-list-across-workspaces}

可以在工作区之间共享和引用列表和智能列表，以便从数据库中轻松重复使用。

>[!NOTE]
>
>应用人员分区规则(智能列表和工作区中的静态列表仅显示同为列表成员的人员 *和* 当前工作区的成员)。

## 共享列表或智能列表  {#share-a-list-or-smart-list}

1. 转到 **数据库**.

   ![](assets/db-1.png)

1. 右键单击营销活动文件夹。 选择 **新建文件夹**.

   >[!NOTE]
   >
   >仅当资产嵌套在文件夹中时，才能在工作区之间共享资产。

   ![](assets/two-4.png)

1. 将要共享的列表或智能列表拖放到新文件夹中。

   ![](assets/three-4.png)

1. 右键单击 **新建文件夹** 选择 **共享文件夹**.

   ![](assets/four-3.png)

1. 选择 **工作区** 共享。 选择 **保存**.

   ![](assets/image2014-12-9-15-3a37-3a25.png)

   太棒了！ 此列表现在将在两个工作区中都可用。

   >[!NOTE]
   >
   >在营销活动中，您只能共享顶级文件夹。 在数据库中，您既可以共享顶级文件夹，也可以共享一个级别级别级别的文件夹。
