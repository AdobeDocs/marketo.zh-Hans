# 键盘快捷键术语表 {#keyboard-shortcuts-glossary}

你是键盘快捷键吗？ 我们掩护你！

## 区域导航： {#area-navigation}

| 键盘快捷键 | 区域 |
|---|---|
| Alt/Opt + A | Analytics |
| Alt/Opt + C | 日历 |
| Alt/Opt + D | Design Studio |
| Alt/Opt + F | 全局搜索 |
| Alt/Opt + L | 数据库 |
| Alt/Opt + M | 营销活动 |
| 选择+ N | 管理员 |
| 选择+ O | 我的Marketo |
| Alt/Opt + S | 下拉超级球菜单 |

## 日历/计划视图导航：  {#calendar-schedule-view-navigation}

| 键盘快捷键 | 操作 |
|---|---|
| Alt/Opt +向上箭头键 | 及时移回 |
| Alt/Opt +向下箭头 | 及时前进 |
| Alt/Opt + T | 转到“今天” |

## 其他： {#miscellaneous}

| 键盘快捷键 | 操作 |
|---|---|
| ESC | 特写弹出窗口 |
| 返回/进入 | 提交弹出窗口 |
| Ctrl + Alt/Opt + H | 在表单编辑器中隐藏/取消隐藏字段 |
