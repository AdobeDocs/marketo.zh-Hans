# 新建Campaign文件夹 {#create-new-campaign-folder}

Campaign文件夹可帮助您保持整洁的工作区。 按照这些简单的步骤操作，您即将实现自动化魔法。

1. 转到 **营销活动**.

   ![](assets/login-marketing-activities.png)

1. 选择 **新建**.

   ![](assets/image2015-2-25-7-3a57-3a18.png)

1. 选择 **新建Campaign文件夹**.

   ![](assets/image2015-2-25-7-3a58-3a15.png)

1. 输入 **名称** ，以查看营销活动文件夹。

   ![](assets/image2015-2-25-8-3a0-3a20.png)

1. 可选：输入 **描述** 单击 **创建**.

   >[!TIP]
   >
   >描述适用于订阅中的其他用户。 您的客户将看不到此消息。

   ![](assets/image2015-2-25-8-3a9-3a3.png)

   太棒了！ 您刚刚构建了一个营销活动文件夹。 在树上看看。

   ![](assets/image2015-2-25-8-3a10-3a29.png)

   现在， [创建新项目](/help/marketo/product-docs/core-marketo-concepts/programs/creating-programs/create-a-program.md)，您将看到此campaign文件夹作为一个选项。
