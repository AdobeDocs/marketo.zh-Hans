# 重命名Marketo资产 {#rename-a-marketo-asset}

文件夹、程序和本地资产都可以轻松重命名。

1. 选择要重命名的任何资产，然后单击右侧面板名称块。

   ![](assets/image2015-4-10-17-19-48.png)

1. 在文本字段中键入新名称。 单击 **保存**.

   ![](assets/image2015-4-10-17-3a19-3a33.png)

   >[!NOTE]
   >
   >您无法重命名已上传的文件和图像，或智能列表或智能营销活动中当前正在引用的资产（在过滤器、触发器、流程步骤等中）。
