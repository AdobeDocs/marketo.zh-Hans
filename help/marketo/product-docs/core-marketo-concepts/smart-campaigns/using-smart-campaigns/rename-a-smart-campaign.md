# 重命名智能活动 {#rename-a-smart-campaign}

您可以更改现有智能活动的名称。 这是方法。

1. 转到营销活动。

   ![](assets/login-marketing-activities.png)

1. 选择您的智能活动，然后单击其名称。

   ![](assets/renamecampaign-hands.png)

   >[!TIP]
   >
   >项目中的智能活动名称始终转换为格式“ProgramName.CampaignName”。

1. 输入新的智能活动名称，然后单击保存。

   ![](assets/rename-cursorandhand.png)

   >[!NOTE]
   >
   >旧名称显示在选项卡中，并在保存时更改。

快速轻松！ 智能活动被引用的任何地方也将被更改。
