# 在智能营销活动中查看被阻止的人员 {#view-blocked-people-in-a-smart-campaign}

Marketo数据库中被阻止的人员列入阻止列表会被标记、取消订阅、暂停营销，和/或者电子邮件无效或为空。

>[!NOTE]
>
>Marketo正在更改黑名单和白名单等术语，以便在我们的产阻止列表品中允许列表和。 在此更新过程中，您可能会在我们的UI和文档屏幕截图中看到旧术语，在我们的文档文本中看到新术语。 我们为任何混淆道歉。

1. 在您的智能营销活动中，单击 **计划**. 在 **智能列表状态** 单击第二个链接。

   ![](assets/image2014-9-22-16-3a47-3a38.png)

   >[!NOTE]
   >
   >达到通信限制的人，也可以被阻止接收邮件。 了解如何 [编辑通信限制](/help/marketo/product-docs/administration/email-setup/enable-communication-limits.md) 中。

   “被阻止的人员”选项卡显示可能未收到邮件的人员。

   ![](assets/image2014-9-22-16-3a48-3a11.png)

   >[!NOTE]
   >
   >操作电子邮件将发送给标记为未订阅和营销暂停的人员。
