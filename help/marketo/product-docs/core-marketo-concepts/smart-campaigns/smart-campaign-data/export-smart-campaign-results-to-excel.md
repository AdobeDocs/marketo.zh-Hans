# 将Smart Campaign结果导出到Excel {#export-smart-campaign-results-to-excel}

将智能营销活动结果导出到Excel以供在Marketo之外使用。

1. 在智能营销活动中，单击 **结果** 然后是底部的导出图标。

   ![](assets/exportexcel-hands.png)

   很快即可导出结果并准备使用！

   >[!NOTE]
   >
   >导出限制为20,000行。 Excel文件将保存在浏览器的默认下载文件夹中。
