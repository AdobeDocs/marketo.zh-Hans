# 将智能营销活动结果导出到Excel {#export-smart-campaign-results-to-excel}

将智能营销活动结果导出到Excel，以便在Marketo外使用。

1. 在您的智能营销活动中，单击 **结果** 然后，在底部显示导出图标。

   ![](assets/exportexcel-hands.png)

   很快，您的结果即已导出并可供使用！

   >[!NOTE]
   >
   >导出限制为20,000行。 Excel文件保存在浏览器的默认下载文件夹中。
