# 从流程中移除 {#remove-from-flow}

假设您有一个Smart Campaign流程，该流程使用“发送警报”来提醒销售代表呼叫热门潜在客户。 它每天发送一条消息，直到销售代表致电。 在联系潜在客户以停止进一步警报后，您可以在触发营销活动中使用“从流量中删除”。 它就像个人的Smart Campaign弹射座椅。

![](assets/remove-from-flow-1.png)

>[!NOTE]
>
>这通常会影响处于营销活动流程等待步骤的人员。

1. 查找并选择要从中删除人员的Smart Campaign。

   ![](assets/remove-from-flow-2.png)

>[!NOTE]
>
>您可以选择特定的智能营销活动，或在&#x200B;**[!UICONTROL Campaign]**&#x200B;下拉列表中选择“此营销活动”以选择您当时实际参与的营销活动。

>[!NOTE]
>
>此功能旨在在Smart Campaign的流程步骤中使用。
