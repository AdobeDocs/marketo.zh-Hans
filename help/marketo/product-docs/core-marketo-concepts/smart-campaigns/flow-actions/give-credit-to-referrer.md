# 将点数给予反向链接 {#give-credit-to-referrer}

## 概述 {#overview}

运行 [推荐选件](/help/marketo/product-docs/demand-generation/social/referral-offers/create-a-referral-offer.md) 或 [抽奖](/help/marketo/product-docs/demand-generation/social/sweepstakes/create-sweepstakes.md)，则可以使用不同方法将点数分给反向链接：

* 引见
* 引荐注册
* **智能列表触发器**
* 自定义JavaScript事件

如果您选择使用 **智能列表触发器** 选项来指定目标，您将需要使用 **将点数给予反向链接** 流步骤。

![](assets/image2014-9-22-15-3a59-3a18.png)

## 使用情况 {#usage}

1. 构建营销活动并决定要触发的操作后，只需查找并选择您要将点数授予反向链接的社交应用程序即可。

   ![](assets/image2014-9-22-15-3a59-3a39.png)

   >[!NOTE]
   >
   >确保将您的社交应用程序配置为使用智能列表触发器。 请参阅  [指定引荐选件的目标](/help/marketo/product-docs/demand-generation/social/referral-offers/specify-goal-for-referral-offer.md) 以了解详细信息。

太棒了！ 通过此流程步骤处理的任何人员现在都将为其反向链接提供点数。
