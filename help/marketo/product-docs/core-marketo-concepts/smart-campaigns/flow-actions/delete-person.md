# 删除人员{#delete-person}

有时，错误的人会进入您的数据库。 “删除人员”流步骤可以删除这些流步骤。

## 概述{#overview}

在智能活动中使用流步骤。

![](assets/one-4.png)

>[!CAUTION]
>
>删除人员时，其所有历史RCE数据也将被删除。 无法撤消。

## 用法{#usage}

当您在流步骤中拖动时，也会自动设置为从您的CRM中删除。

![](assets/two-4.png)

您可以从Marketo中删除，而不能从CRM中删除，如下所示：

![](assets/three-3.png)

>[!NOTE]
>
>从CRM **中删除人员仅适用于Salesforce**。 如果从Marketo中删除人员并选择将其保留在Salesforce中，则如果其Salesforce记录有更新，则会在Marketo中重新创建他们。
