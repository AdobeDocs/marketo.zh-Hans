# 添加到参与项目 {#add-to-engagement-program}

通过此流程步骤构建的Smart Campaign将成为参与计划的网关。

![](assets/add-to-engagement-program-1.png)

1. 选择要将人员添加到的参与计划。

   ![](assets/add-to-engagement-program-2.png)

1. 选择您希望放置人员的流。

   ![](assets/add-to-engagement-program-3.png)

   >[!NOTE]
   >
   >您不能将人员添加到同一项目群内的多个流中。
