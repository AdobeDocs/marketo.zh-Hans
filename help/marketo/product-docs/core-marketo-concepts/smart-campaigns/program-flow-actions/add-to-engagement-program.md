# 添加到“参与”项目{#add-to-engagement-program}

## 概述{#overview}

使用此流程步骤构建的智能活动将成为您的参与项目的门户。

![](assets/image2014-9-22-14-3a47-3a32.png)

## 用法{#usage}

1. 选择要将人员添加到的参与项目。

   ![](assets/image2014-9-22-14-3a47-3a36.png)

1. 选择要将人员放入的流。

   ![](assets/image2014-9-22-14-3a47-3a39.png)

   >[!NOTE]
   >
   >不能在同一项目内将人员添加到多个流。
