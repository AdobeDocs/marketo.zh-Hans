# 更改项目状态 {#change-program-status}

## 概述 {#overview}

项目状态可帮助您跟踪人员通过项目或事件的进展情况。 欲知更多信息，请访问 [自定义、创建和管理渠道](/help/marketo/product-docs/administration/tags/create-a-program-channel.md).

>[!CAUTION]
>
>更改参与计划中的计划状态会自动将其添加到第一个流中。 他们将开始接收内容。

## 使用情况 {#usage}

1. 拖入 **更改项目状态** 流程步骤。

   ![](assets/image2014-9-22-14-3a43-3a34.png)

1. 选择 **新状态** 您想要设置。 此人如果尚未成为计划成员，也将成为计划成员。

   ![](assets/image2014-9-22-14-3a43-3a45.png)

选项仅限于该程序的有效状态。

>[!NOTE]
>
>人员无法按照“管理员”中的渠道编辑器的定义，向后移动到以前的项目状态。

状态是用于跟踪人员和报告的强大工具。 这值得你的时间。
