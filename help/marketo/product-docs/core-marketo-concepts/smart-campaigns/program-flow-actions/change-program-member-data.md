# 更改项目成员数据 {#change-program-member-data}

您可以使用Marketo通过利用“更改数据值”流程操作来更新字段的值。

>[!NOTE]
>
>您还可以阻止字段更新。 有关详细信息，请参阅阻止字段更新。

1. 在Smart Campaign的“流量”选项卡中，引入&#x200B;**[!UICONTROL Change Program Member Data]**&#x200B;流量步骤并选择所需的程序。

   ![](assets/change-program-member-data-1.png)

1. 查找并选择要更改其值的属性。

   ![](assets/change-program-member-data-2.png)

1. 输入所需的属性值。

   ![](assets/change-program-member-data-3.png)

>[!NOTE]
>
>您还可以在[!UICONTROL New Value]中使用令牌。

准备就绪后运行Smart Campaign。

>[!TIP]
>
>如果要清除字段而不是更新它们，可以输入“NULL”（无引号，全部大写）作为[!UICONTROL New Value]。
