# 将人员同步到Microsoft {#sync-person-to-microsoft}

此功能仅适用于将Marketo Engage与集成的人员 [!DNL Microsoft Dynamics].

## 概述 {#overview}

此流程步骤会将Marketo创建的人员插入到您的Dynamics CRM。

![](assets/one.png)

## 使用情况 {#usage}

您可以将Dynamics用户设置为人员所有者。
