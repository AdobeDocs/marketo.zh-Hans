# 在Microsoft中更改所有者 {#change-owner-in-microsoft}

如果您的现有人员已经分配给所有者，则可以使用此流程步骤将他们重新分配给其他所有者。

>[!NOTE]
>
>此流程步骤将 **仅当与触发器一起使用时才可用**，而不是过滤器。

![](assets/one-1.png)

**使用情况**

1. 只需选择要改为的所有者，然后离开！

   ![](assets/two-1.png)

   >[!NOTE]
   >
   >如果您的Dynamics帐户中尚不存在该记录，我们将同步该记录，然后将其分配给选定的用户。
