# 在Microsoft中更改所有者 {#change-owner-in-microsoft}

如果您现有的人员已分配给所有者，则可以使用此流程步骤将它们重新分配给其他所有者。

>[!NOTE]
>
>此流程步骤将 **仅当与触发器一起使用时才有效**，而不是过滤器。

![](assets/one-1.png)

**使用情况**

1. 只需选择要更改为的所有者即可转到！

   ![](assets/two-1.png)

   >[!NOTE]
   >
   >如果记录尚未在您的Dynamics帐户中存在，我们会将其同步到此处，然后将其分配给选定的用户。
