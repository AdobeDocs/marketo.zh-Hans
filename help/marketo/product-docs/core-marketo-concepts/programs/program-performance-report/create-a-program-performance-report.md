# 创建项目性能报告{#create-a-program-performance-report}

通过此报告了解项目的出色统计数据。

1. [在程序中创建报](/help/marketo/product-docs/reporting/basic-reporting/creating-reports/create-a-report-in-a-program.md) 告，然后选择 **项目** [性能报告类型](/help/marketo/product-docs/reporting/basic-reporting/report-types/report-type-overview.md)。
1. 单击&#x200B;**报告**&#x200B;选项卡。
1. 你在那！ 浏览您的报告，了解项目的表现。

   ![](assets/image2014-9-18-17-3a23-3a2.png)

>[!TIP]
>
>例如，要查找&#x200B;*每成功成本*&#x200B;最低的项目，请对该列的报表](/help/marketo/product-docs/reporting/basic-reporting/editing-reports/sort-report-on-columns.md)排序，然后选择&#x200B;**升序排序**。[

项目性能报告中的列包括：

<table> 
 <thead> 
  <tr> 
   <th>列</th> 
   <th>说明</th> 
  </tr> 
 </thead> 
 <tbody> 
  <tr> 
   <td>渠道</td> 
   <td>渠道的项目类型。</td> 
  </tr> 
  <tr> 
   <td>新名称</td> 
   <td>新人被项目获取。</td> 
  </tr> 
  <tr> 
   <td>成功</td> 
   <td>在项目状态中定义的成功成员数。 </td> 
  </tr> 
  <tr> 
   <td>总成本</td> 
   <td><p>项目内所有期间成本的总和。</p></td> 
  </tr> 
 </tbody> 
</table>
