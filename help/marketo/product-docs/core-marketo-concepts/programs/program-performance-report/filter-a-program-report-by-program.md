# 按项目筛选项目报告 {#filter-a-program-report-by-program}

将[项目绩效报表](/help/marketo/product-docs/core-marketo-concepts/programs/program-performance-report/create-a-program-performance-report.md){target="_blank"}的重点放在特定项目上，以比较其绩效。

1. 转到&#x200B;**[!UICONTROL Marketing Activities]**（或&#x200B;**[!UICONTROL Analytics]**）。

   ![](assets/login-marketing-activities-3.png)

1. 选择您的项目性能报表。

   ![](assets/image2014-9-23-16-3a4-3a4.png)

1. 单击&#x200B;**[!UICONTROL Setup]**&#x200B;选项卡并拖到&#x200B;**[!UICONTROL Programs]**&#x200B;上。

   ![](assets/prospect3.jpg)

1. 选择要包含在报表中的文件夹和特定程序。

   ![](assets/image2014-9-23-16-3a5-3a5.png)

   >[!TIP]
   >
   >如果选择文件夹，则报表将包含该文件夹在报表运行时包含的所有内容。

1. 仅此而已！ 单击“**[!UICONTROL Report]**”选项卡，查看报告中的&#x200B;_仅_&#x200B;个选定程序。

   ![](assets/image2014-9-23-16-3a5-3a41.png)
