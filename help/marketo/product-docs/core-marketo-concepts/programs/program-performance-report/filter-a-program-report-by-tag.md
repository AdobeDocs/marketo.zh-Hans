# 按标记{#filter-a-program-report-by-tag}筛选项目报告

将您的[项目性能报告](/help/marketo/product-docs/core-marketo-concepts/programs/program-performance-report/create-a-program-performance-report.md)重点放在特定[标签](/help/marketo/product-docs/core-marketo-concepts/programs/working-with-programs/understanding-tags.md)上。

1. 转至&#x200B;**Marketing** **活动**（或&#x200B;**Analytics**）。

   ![](assets/login-marketing-activities.png)

1. 选择&#x200B;**项目性能**&#x200B;报告。

   ![](assets/image2014-9-23-16-3a12-3a36.png)

1. 单击&#x200B;**设置**&#x200B;选项卡，并拖动到&#x200B;**标记**&#x200B;过滤器中的一个上方。

   ![](assets/prospects.jpg)

1. 选择要包含在报表中的标记值。

   ![](assets/prospect1.jpg)

1. 单击&#x200B;**应用**。

   ![](assets/prospect2.jpg)

1. 完成! 单击&#x200B;**报表**&#x200B;选项卡，查看&#x200B;_仅_&#x200B;与报表中选定标记匹配的项目。

   ![](assets/image2014-9-23-16-3a14-3a42.png)

   >[!NOTE]
   >
   >[按期间成本筛选项目报表](/help/marketo/product-docs/core-marketo-concepts/programs/program-performance-report/filter-a-program-report-by-period-cost.md)
