## 冲突规则 {#conflict-rules}

* **项目标记**
   * 在此订阅中创建标记（默认） - _推荐_
   * 忽略

* **同名登陆页面模板**
   * 复制原始模板（默认） —  _推荐_
   * 使用目标模板

* **具有相同名称的图像**
   * 保留两个文件（默认） —  _推荐_
   * 替换此订阅中的项目

* **同名电子邮件模板**
   * 保留两个模板（默认） —  _推荐_
   * 替换现有模板
