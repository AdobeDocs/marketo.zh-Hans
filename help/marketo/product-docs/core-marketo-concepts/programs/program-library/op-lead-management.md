# 业务线索管理 {#op-lead-management}

这是商机管理最佳实践工作流的示例，该工作流利用Marketo Engage默认程序协助您管理Marketo Engage数据库中对CRM的记录。

>[!NOTE]
>
>在Marketo Engage中，数据库中的记录称为人员/人员。 此示例中的商机管理是引用CRM中的记录。

要获得进一步的战略帮助或自定义计划，请联系Adobe客户团队或访问[Adobe Professional Services](https://business.adobe.com/customers/consulting-services/main.html)页面。

## 冲突规则 {#conflict-rules}

* **项目标记**
   * 在此订阅中创建标记 — _推荐_
   * 忽略

* **同名登陆页面模板**
   * 复制原始模板 — _推荐_
   * 使用目标模板

* **具有相同名称的图像**
   * 保留两个文件 — _推荐_
   * 替换此订阅中的项目

* **同名电子邮件模板**
   * 保留两个模板 — _推荐_
   * 替换现有模板

## 最佳实践 {#best-practices}

* 请考虑添加其他智能营销活动，以满足您可能在组织中跟踪的每个生命周期状态需求。 此项目中构建的每个营销活动都旨在作为最佳实践构建的一个示例，而不是特定于所有用例。 请记住要更新Smart Campaigns以满足您特定的潜在客户生命周期管理流程的需要。

* 请考虑更新此程序示例的命名规则以与您的命名规则一致。
