# 在项目计划视图中创建自定义叠加图 {#creating-custom-overlays-in-program-schedule-view}

您可以创建自定义叠加以查看与您的需求相关的条目。

1. 单击 **日程** 下拉菜单。

   ![](assets/image2014-9-24-10-3a20-3a11.png)

1. 选择 **叠加**.

   ![](assets/image2014-9-24-10-3a20-3a17.png)

1. 选择要在叠加中可见的条目类型。

   ![](assets/image2014-9-24-10-3a20-3a26.png)

1. 您还可以按以下项过滤 [项目标记](/help/marketo/product-docs/core-marketo-concepts/programs/working-with-programs/understanding-tags/use-tags-in-a-program.md).

   ![](assets/image2014-9-24-10-3a20-3a32.png)

   太棒了。 现在，您的叠加将仅显示您定义的条目。

   ![](assets/image2014-9-24-10-3a20-3a37.png)
