# 浏览项目群计划视图 {#navigating-the-program-schedule-view}

以下是帮助您浏览项目计划视图的基础知识。

## 查找计划视图 {#find-the-schedule-view}

1. 转到 **营销活动**.

   ![](assets/login-marketing-activities.png)

1. 选择您的项目。 单击 **视图** 下拉菜单。 选择 **计划**.

   ![](assets/image2014-9-17-11-3a38-3a3.png)

   现在，您将查看项目的计划视图。

   ![](assets/image2014-9-17-11-3a38-3a14.png)

>[!NOTE]
>
>项目计划视图具有粘性。 设置后，所有程序都将默认使用计划视图。

## 在条目之间切换 {#switching-between-entries}

1. 在条目详细信息中，单击箭头以移动到下一个计划条目。

   ![](assets/image2014-9-17-11-3a38-3a54.png)

   很酷吧？

   ![](assets/image2014-9-17-11-3a39-3a10.png)

## 查看上下文菜单 {#view-context-menu}

1. 右键单击任何项目群以编辑项目、智能列表、设置、我的令牌或成员。

   ![](assets/image2014-9-17-11-3a39-3a59.png)

## 在模式之间更改 {#changing-between-modes}

1. 单击 **3周** 或 **月** 将更改显示的可见日期。

   ![](assets/image2014-9-17-11-3a40-3a19.png)

## 全屏视图 {#full-screen-view}

1. 您可以单击右上角的屏幕图标，以全屏模式查看项目计划。

   ![](assets/image2014-9-17-11-3a40-3a45.png)

太棒了！ 现在您已经知道如何查看程序，让我们学习它还可以执行的其他有趣操作。
