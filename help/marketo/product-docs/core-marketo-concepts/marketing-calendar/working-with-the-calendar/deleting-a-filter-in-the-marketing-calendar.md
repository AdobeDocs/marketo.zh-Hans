# 在营销日历中删除筛选器 {#deleting-a-filter-in-the-marketing-calendar}

如果您希望删除营销日历中的筛选器，您已到达正确的位置。

1. 选择要删除的筛选器。

   ![](assets/image2014-9-24-11-3a27-3a32.png)

1. 单击红色 **x**。

   ![](assets/image2014-9-24-11-3a27-3a36.png)

1. 单击 **删除** ，以进行确认。

   ![](assets/image2014-9-24-11-3a27-3a42.png)

   再见。
