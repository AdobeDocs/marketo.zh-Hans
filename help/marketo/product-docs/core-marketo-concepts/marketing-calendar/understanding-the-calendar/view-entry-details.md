# 查看条目详情 {#view-entry-details}

1. 从日历中选择一个条目。

   ![](assets/image2014-9-26-10-3a30-3a44.png)

1. 登录在营销日历中是只读的。 导航到项目以进行编辑。

   ![](assets/image2014-9-26-10-3a31-3a1.png)

>[!TIP]
>
>右键单击详细信息面板可访问导航菜单或打开编辑器。
