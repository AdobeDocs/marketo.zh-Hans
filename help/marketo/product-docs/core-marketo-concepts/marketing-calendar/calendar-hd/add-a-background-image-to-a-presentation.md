# 将背景图像添加到演示文稿 {#add-a-background-image-to-a-presentation}

通过选择背景图像自定义演示文稿。

>[!PREREQUISITES]
>
>[创建演示文稿](/help/marketo/product-docs/core-marketo-concepts/marketing-calendar/calendar-hd/create-a-presentation.md)

1. 右键单击演示文稿并选择 **查看设置**.

   >[!NOTE]
   >
   >您还可以双击演示文稿以进入设置选项卡。

   ![](assets/image2015-3-24-14-3a36-3a52.png)

1. 拖放 **背景图像** 从右树到画布。

   ![](assets/image2015-3-24-14-3a39-3a40.png)

1. 从图像库中选择图像。

   >[!TIP]
   >
   >为了获得最清晰的外观，请使用 **1920 x 1080** 或 **1280 x 720**.

   ![](assets/image2015-3-24-14-3a47-3a57.png)

1. 单击 **预览** 来检查一下。

   ![](assets/image2015-3-24-14-3a51-3a1.png)

   就这样！
