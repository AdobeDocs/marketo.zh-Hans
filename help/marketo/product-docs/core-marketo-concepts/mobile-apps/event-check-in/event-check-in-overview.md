# 事件签入概述{#event-check-in-overview}

适用于iPad或Android平板电脑的Marketo事件应用程序使得到达您的事件时立即登记用户变得很容易。

本文概述了整个过程。 标题链接到单独的文章，某些Marketo活动是此大纲中的链接。

在事件之前会发生一些活动，如创建新角色和邀请与会者。 在事件期间会发生一些情况，如签入与会者。 事件后，您可以将数据与Marketo同步，并更新人员的状态。

祝您事件!

## 准备您的事件

事件前：

* [为iPad或Android用](/help/marketo/product-docs/core-marketo-concepts/mobile-apps/event-check-in/grant-users-access-to-the-check-in-app.md) 户为事件新建Marketo Role
* [邀请新的平板电脑用户参](/help/marketo/product-docs/core-marketo-concepts/mobile-apps/event-check-in/grant-users-access-to-the-check-in-app.md) 加活动并为他们分配新角色
* [新建Marketo 事件项目](/help/marketo/product-docs/demand-generation/events/understanding-events/create-a-new-event-program.md)
* 指定事件日期
* 将人物添加到事件

## 设置iPad或Android平板电脑

要准备iPad或Android平板电脑，请执行以下操作：

* 下载并安装Marketo 事件 iPad或Android平板电脑应用程序
* 确认登录凭据
* 同步应用程序以填充事件
* 确认签入正在工作

## 使用iPad或平板电脑检查人员

事件:

* 登录到应用程序
* 查找客人（并创建新人）
* [入住客人](/help/marketo/product-docs/core-marketo-concepts/mobile-apps/event-check-in/check-people-into-your-event-from-your-tablet.md) （反向登记，在入住时编辑个人记录）
* 将iPad或Android平板电脑应用程序与Marketo同步
