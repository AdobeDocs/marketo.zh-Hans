# 个性化Marketo时刻 {#personalizing-marketo-moments}

当有很多营销项目和明智的营销活动在进行时，能看到所有内容会很棒，但您可能只想专注于自己的工作。

启用 **仅显示我的时刻** 以仅显示您自己的电子邮件项目和智能营销活动。

![](assets/image2015-7-16-15-3a53-3a24.png)

或者，禁用 **仅显示我的时刻** 查看您有权访问的所有Smart Campaign和电子邮件程序。

![](assets/image2015-7-16-15-3a55-3a29.png)
