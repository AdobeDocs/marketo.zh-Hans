# 个性化营销时刻 {#personalizing-marketo-moments}

当您有大量营销项目和智能活动在进行时，了解一切会很棒，但您可能希望只专注于自己的工作。

启用 **仅显示我的时刻** ，以仅显示您自己的电子邮件项目和智能活动。

![](assets/image2015-7-16-15-3a53-3a24.png)

或者，禁 **用“仅显示我的时** 刻”来视图您有权访问的所有智能活动和电子邮件项目。

![](assets/image2015-7-16-15-3a55-3a29.png)
