# 个性化Marketo时刻{#personalizing-marketo-moments}

当您有许多营销项目和智能活动在进行时，了解一切都会很好，但您可能希望只关注您自己的工作。

启用&#x200B;**仅显示我的瞬间**&#x200B;以仅显示您自己的电子邮件项目和智能活动。

![](assets/image2015-7-16-15-3a53-3a24.png)

或者，禁用&#x200B;**仅显示我的瞬间**&#x200B;以视图您有权访问的所有智能活动和电子邮件项目。
