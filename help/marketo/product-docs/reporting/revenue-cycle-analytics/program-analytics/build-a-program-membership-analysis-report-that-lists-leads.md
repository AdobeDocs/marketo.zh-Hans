# 构建项目会员资格分析报告，让列表主导 {#build-a-program-membership-analysis-report-that-lists-leads}

请按照以下步骤创建项目会员分析报告，该报告将显示所有项目会员姓名。

>[!NOTE]
>
>**可用性**
>
>并非所有客户都购买了此功能。 有关详细信息，请与销售代表联系。

1. 启动收入浏览器。

   ![](assets/one.png)

1. 依次单击新建和报告。

   ![](assets/two.png)

1. 选择项目会员资格分析，然后单击确定。

   ![](assets/three.png)

1. 查找并多次单击项目名黄点。

   ![](assets/four.png)

1. 查找并多次单击“成员”蓝点。

   ![](assets/five.png)

1. 查找并多次单击成功（总数）蓝点。

   ![](assets/six.png)

看这有多简单？

![](assets/seven.png)

您可以快速添加潜在客户属性 **(如“全名** ”)，以查看成员。

>[!TIP]
>
>有许多潜在客户/公司属性可供挑选。 看看他们！

1. 查找并多次单击“全名”黄点。

   ![](assets/eight.png)

给你！

![](assets/nine.png)
