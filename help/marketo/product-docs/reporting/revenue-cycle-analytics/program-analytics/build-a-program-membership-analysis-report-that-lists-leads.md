# 构建列出潜在客户的计划会员资格分析报告 {#build-a-program-membership-analysis-report-that-lists-leads}

按照以下步骤创建项目群成员资格分析报告，该报告将显示所有项目群成员名称。

>[!AVAILABILITY]
>
>并非所有Marketo Engage用户都已购买此功能。 有关详细信息，请联系Adobe客户团队（您的客户经理）。

1. 启动Revenue Explorer。

   ![](assets/one.png)

1. 单击&#x200B;**新建**，然后单击&#x200B;**报告**。

   ![](assets/two.png)

1. 选择&#x200B;**计划成员资格分析**，然后单击&#x200B;**确定**。

   ![](assets/three.png)

1. 查找并双击程序名称黄色圆点。

   ![](assets/four.png)

1. 查找并双击成员蓝点。

   ![](assets/five.png)

1. 找到并双击Success (Total)(成功（总计）)蓝点。

   ![](assets/six.png)

看看这有多容易？

![](assets/seven.png)

您可以快速添加潜在客户属性，如&#x200B;_全名_，以查看成员是谁。

>[!TIP]
>
>有许多潜在客户/公司属性可供选择。 看看他们！

1. 查找并双击&#x200B;**全名**&#x200B;黄色圆点。

   ![](assets/eight.png)

拿着！

![](assets/nine.png)
