# 了解收入浏览器{#understanding-opportunity-analysis-in-revenue-explorer}中的业务机会分析

收入浏览器中的业务机会分析使您能够更深入地检查您的业务机会。 根据任何潜在客户/公司属性（如潜在客户来源、行业或地理位置）对数据进行切片和分割。 根据名称、阶段或概率分析机会的创建和关闭。 了解营销对渠道的贡献。

## 示例分析{#example-analysis}

以下是您可以在“业务机会”分析区创建的一些报表。

1. 营销对创造机会的影响

   您的公司渠道中受营销项目影响的百分比是多少？ 这份报告给了你答案。 以下饼图显示了营销项目获得的所有机会的百分比和机会金额。

   ![](assets/image2015-7-21-16-3a21-3a4.png)

1. 营销对商机的影响已关闭并成功。

   此报表按数量和金额显示营销获取和影响的收入。

   ![](assets/image2015-7-21-16-3a41-3a55.png)

1. 由潜在客户来源关闭的机会

   此报表将按潜在客户来源分类所有已关闭的商机，并为您清晰概述哪些来源有效以及哪些来源无效。

   ![](assets/image2015-7-21-10-3a34-3a50.png)

1. 按源关闭时间

   此报表说明了关闭机会的平均天数与潜在客户来源之间的关系。

   ![](assets/image2015-7-21-10-3a35-3a3.png)

1. Open Opportunity and Stage

   此报表显示每个收入周期阶段中打开的机会数。

   ![](assets/image2015-7-21-10-3a35-3a32.png)

1. 按行业分列的机会数

   这份报告回答了这样的问题：“我们每年从某些行业获得的机会是否越来越多？”

   ![](assets/image2015-7-21-10-3a35-3a45.png)

## 机会分析Dimension和度量{#opportunity-analysis-dimensions-and-measures}

机会分析使您能够访问所有与潜在客户、公司和机会相关的维度以及与机会相关的衡量指标。 使用这些机会分析维度和度量来回答报表中的特定问题。

1. 公司属性

   | Dimension | 说明 |
   |---|---|
   | 年收入 | 公司的年收入 |
   | 城市 | 公司所在的城市 |
   | 国家/地区 | 公司所在的县 |
   | 行业 | 公司所在的行业 |
   | 公司名称 | 公司的名称 |
   | 员工人数 | 公司中的员工数 |
   | 邮政编码 | 公司 |
   | SIC代码 | 公司的SIC代码 |
   | 州 | 公司所在的状态 |

1. 潜在客户属性

   | Dimension | 说明 |
   |---|---|
   | 列入阻止列表 | 潜在客户已列入阻止列表 |
   | 已转换为Opportunity | 潜在客户被转换为业务机会 |
   | 电子邮件无效 | 潜在客户是否具有有效的电子邮件地址 |
   | 暂停营销 | 潜在客户是否因营销电子邮件而暂停 |
   | 电子邮件地址 | 潜在客户的电子邮件地址 |
   | 职务 | Lead的职务 |
   | 全名 | Lead的全名 |
   | 原始源类型 | 潜在客户的原始源类型 |
   | 注册源类型 | 潜在客户的注册源类型 |
   | 潜在客户所有者电子邮件地址 | 潜在客户所有者的电子邮件地址 |
   | 潜在客户所有者职务 | 潜在客户所有者的职务 |
   | 潜在客户所有者名称 | 潜在客户所有者的姓名 |
   | 潜在客户源 | 潜在客户源 |
   | 潜在客户状态 | 潜在客户状态 |

1. 潜在客户创建时间范围

   | Dimension | 说明 |
   |---|---|
   | 潜在客户创建年份 | 创建潜在客户的年份 |
   | 潜在客户创建季度 | 创建潜在客户的季度 |
   | 潜在客户创建月 | 创建潜在客户的月份 |
   | 潜在客户创建周 | 创建潜在客户的周 |
   | 潜在客户创建日期 | 创建潜在客户的日期 |

1. 机会属性

   | Dimension | 说明 |
   |---|---|
   | 已结业 | 机会是否已关闭 |
   | 业务机会预测类别 | 机会预测类别 |
   | 业务机会名称 | 机会名称 |
   | 机会阶段 | 机会阶段 |
   | 机会类型 | 机会类型 |
   | 赢得机会 | 这个机会是否已经结束，并且赢得 |
   | 营销影响的机会 | 此标记指示在任何营销项目中是否已获得或已成功获取任何潜在客户/联系人。 只考虑定义了期间成本的项目。 |

1. Opportunity Closed时间范围

   | Dimension | 说明 |
   |---|---|
   | 已结业年 | 机会关闭的年份 |
   | 已结业 | 机会关闭的季度 |
   | 已结月 | 机会关闭的月份 |
   | 已结业周 | 机会关闭的那周 |
   | 业务机会结束日期 | 业务机会关闭的日期 |

1. Opportunity Created时间范围

   | Dimension | 说明 |
   |---|---|
   | 创建的业务机会年 | 创建机会的年份 |
   | 业务机会创建季度 | 创建业务机会的季度 |
   | 业务机会创建月 | 创建业务机会的月份 |
   | Opportunity Created Week | 创建业务机会的周 |
   | 业务机会创建日期 | 创建业务机会的日期 |

1. 措施

   | 度量 | 说明 |
   |---|---|
   | 结束机会的平均天数 | 结束业务机会的平均天数 |
   | 结束销售机会的平均天数（丢失） | 丢失机会的平均天数 |
   | 关闭业务机会的平均天数（韩元） | 机会的平均天数 |
   | 所有机会的数量 | 所有机会的总数 |
   | 机会数（已关闭） | 已关闭（赢或输）的机会总数 |
   | 机会数（丢失） | 丢失的机会总数 |
   | 机会数（开放） | 仍在打开的机会总数 |
   | 机会数（韩元） | 赢得的机会总数 |
   | 机会金额 | 总机会金额。 如果多个潜在客户与一个业务机会关联，则分配金额将基于潜在客户得分。 |
   | 机会金额（损失） | 损失机会的总额。 如果多个潜在客户与一个业务机会关联，则分配金额将基于潜在客户得分。 |
   | 业务机会金额（未结） | 未结业务机会的总金额。 如果多个潜在客户与一个业务机会关联，则分配金额将基于潜在客户得分。 |
   | 机会金额（韩元） | 赢得的机会的总金额。 如果多个潜在客户与一个业务机会关联，则分配金额将基于潜在客户得分。 |
