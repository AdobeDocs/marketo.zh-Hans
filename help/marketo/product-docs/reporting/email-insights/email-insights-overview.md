# 电子邮件分析概述 {#email-insights-overview}

电子邮件分析可以为电子邮件营销人员提供来自历史数据的强大分析。 它包含两个单独但关联的部分：Analytics和发送。

要访问电子邮件分析，只需在我的Marketo（或Analytics主屏幕）中单击其图块即可。

![](assets/icon.png)

>[!NOTE]
>
>默认情况下，您将看到您有权访问的每个工作区的数据。 您可以取消选择不想查看的工作区。 您的选择将在以后被记住。

## Analytics {#analytics}

使用 [Analytics](/help/marketo/product-docs/reporting/email-insights/email-insights-analytics-overview.md) 以探索电子邮件投放和参与量度的汇总时间序列数据。

![](assets/emailanalytics.jpg)

**过滤器**

已使用高级 [过滤器](/help/marketo/product-docs/reporting/email-insights/filtering-in-email-insights.md) 按受众、内容或平台对结果进行划分。 相同的过滤器同时适用于Analytics和“发送”。

![](assets/filter.png)

**快速图表**

[快速图表](/help/marketo/product-docs/reporting/email-insights/email-insights-quick-charts.md) 是您创建和保存的缩略图，提供了对您而言非常重要的图表概览。

![](assets/three.png)

## 发送 {#sends}

在 [发送](/help/marketo/product-docs/reporting/email-insights/email-insights-sends-overview.md) 页面，检查最近电子邮件通信的特性。

![](assets/two.png)

**量度**

单击量度栏可将颜色转换为数字。

![](assets/delivery-metrics.png)

>[!NOTE]
>
>数据可能最多需要8小时才能在电子邮件分析中反映。

## 自定义Dimension {#custom-dimensions}

包含所有标准Marketo维度，但您可以选择最多添加10个自定义维度。 [自定义维度](/help/marketo/product-docs/reporting/email-insights/custom-dimensions-for-email-insights.md) 包括分段和程序标记。
