# 性能分析贡献概述 {#performance-insights-contribution-overview}

在Marketo性能分析中，默认情况下将显示“贡献”视图。

![](assets/one-1.png)

选择要作为性能查看依据的指标。 在本例中，我们将查看通过收入仪表板中的多点触控赢得的商机。

![](assets/2.png)

>[!NOTE]
>
>了解 [首次接触和多点接触](/help/marketo/product-docs/reporting/revenue-cycle-analytics/revenue-tools/attribution/understanding-attribution.md).

选择您希望查看量度的时段。 在此示例中，我们查看的是当前年份（年初至今）。

![](assets/3-1.png)

>[!NOTE]
>
>我们暂时删除了“上一年”的选定内容。 您仍然可以选择使用自定义范围选择查看整年的性能数据。

量度通过两个图表呈现：圆环图和条形图。

![](assets/four.png)

圆环图显示所选量度的前十个渠道。

![](assets/5-1.png)

条形图显示所选量度在所有渠道中的项目绩效（一次十个项目）。 要查看更多内容，请单击右侧的箭头以滚动到下一组。

![](assets/six.png)

>[!TIP]
>
>如果您希望图形中的条在滚动组时放大，请选择 **按比例缩放Y轴以适合** 复选框。

将鼠标悬停在栏上可查看其他详细信息。

![](assets/seven.png)

在圆环图中选择一个或多个渠道，与这些渠道关联的所有程序都会显示在右侧的条形图中。 再次单击渠道可取消选择。

![](assets/eight.png)

以下数据网格的功能类似于电子表格，显示所选归因模型（首次联系/多点联系）下的所有可用量度。 包含所选量度的列会突出显示。

![](assets/9.png)

| **赢得的机会** | 因影响成功的机会而收到的计划信用部分（以数值表示） |
|---|---|
| **赢得的收入** | 计划因影响赢得的机会而获得的信用部分（以货币价值表示） |
| **成本** | 计划总成本 |
| **每个赢得的机会的成本** | 项目成本与项目因影响创造新机会而获得的信贷部分（以数值表示）的比率 |
| **收入与成本比率** | 为影响赢得的机会和计划成本而获得计划的信贷部分（以货币价值表示）的比例 |

展开一个渠道可查看其排名前十的节目，其余的节目则合并在一起。

![](assets/10.png)

>[!NOTE]
>
>单击上面圆环图中的渠道激活/停用旁边的复选框。
