# 订阅基本报表 {#subscribe-to-a-basic-report}

要接收基本报表的自动更新或共享这些更新，您可以订阅现有报表的任何电子邮件地址。

>[!NOTE]
>
>有关收入周期浏览器报表订阅，请参阅 [订阅收入浏览器报表](/help/marketo/product-docs/reporting/revenue-cycle-analytics/revenue-explorer/subscribe-to-a-revenue-explorer-report.md).

1. 转到 **营销活动** 的上界。

   ![](assets/image2014-9-16-10-3a31-3a54.png)

1. 从导航树中选择您的报表，然后单击 **订阅** 选项卡。

   ![](assets/image2014-9-16-10-3a32-3a1.png)

   >[!NOTE]
   >
   >您还可以订阅 **Analytics** 选项卡。

1. 单击 **新报表订阅**.

   ![](assets/image2014-9-16-10-3a32-3a24.png)

1. 输入电子邮件地址并设置报表电子邮件的频率。

   ![](assets/image2014-9-16-10-3a32-3a31.png)

   >[!NOTE]
   >
   >任何人都可以在收到的电子邮件中取消订阅报表。

   就这样！ 查看您的收件箱！

   ![](assets/image2014-9-16-10-3a32-3a49.png)

   >[!MORELIKETHIS]
   >
   >了解如何 [管理所有报表订阅](/help/marketo/product-docs/reporting/basic-reporting/report-subscriptions/manage-report-subscriptions.md) 在一个地方。
