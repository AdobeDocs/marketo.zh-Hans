# 克隆报告 {#clone-a-report}

创建报告的副本以进一步自定义它而不更改原始报告。

1. 转到 **营销活动** (或 **分析**)区域。

   ![](assets/image2014-9-16-14-3a23-3a46.png)

1. 选择您的报表。 选择 **报表操作** > **克隆报告**.

   ![](assets/image2014-9-16-14-3a23-3a53.png)

   >[!TIP]
   >
   >您还可以通过在树中右键单击报告来克隆报告。

1. 命名您的报表克隆。

   ![](assets/image2014-9-16-14-3a23-3a57.png)

   就是这样 — 您已经准备好开始自定义克隆了！
