# 删除报告{#delete-a-report}

开始[创建报表](/help/marketo/product-docs/reporting/basic-reporting/creating-reports/create-a-report-in-a-program.md)后，您很快就会发现许多报表。 请记住删除不再需要的报告。

1. 右键单击树中不需要的报告，然后选择&#x200B;**删除报告**。

   ![](assets/image2014-9-16-14-3a26-3a48.png)

1. 确认您打算删除该报表。

   ![](assets/image2014-9-16-14-3a26-3a53.png)

   报告从树中消失。 继续，立即删除一些旧报告！
