# 删除报告 {#delete-a-report}

一旦您开始 [创建报告](/help/marketo/product-docs/reporting/basic-reporting/creating-reports/create-a-report-in-a-program.md)很快就能找到很多这样的人。 请记住删除不再需要的报表。

1. 右键单击树中不需要的报告，然后选择 **删除报告**.

   ![](assets/image2014-9-16-14-3a26-3a48.png)

1. 确认您删除报表的意图。

   ![](assets/image2014-9-16-14-3a26-3a53.png)

   报告从树中消失。 继续，立即删除一些旧报告！
