# 删除报告{#delete-a-report}

开始[创建报表](../../../../product-docs/reporting/basic-reporting/creating-reports/create-a-report-in-a-program.md)后，您很快就会发现许多报表。 请记住删除不再需要的报告。

1. 右键单击树中不需要的报告，然后选择&#x200B;**删除报告**。

   ![](assets/image2014-9-16-14-3a26-3a48.png)

1. 确认您打算删除该报表。

   ![](assets/image2014-9-16-14-3a26-3a53.png)

   报告从树中消失。 继续，立即删除一些旧报告！

   >[!NOTE]
   >
   >**深潜**
   >
   >
   >在[基本报告](https://docs.marketo.com/display/docs/basic+reporting)深入了解报表的其他功能。
