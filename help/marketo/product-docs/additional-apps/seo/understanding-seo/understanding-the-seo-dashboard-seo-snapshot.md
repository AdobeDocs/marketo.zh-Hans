# 了解SEO仪表板:SEO快照{#understanding-the-seo-dashboard-seo-snapshot}

使用该仪表板，可以更深入地视图关键词排名的趋势，以及您的网站针对SEO的优化程度。

要了解要重点改进您的SEO工作，请使用此仪表板的[Recommendations](/help/marketo/product-docs/additional-apps/seo/understanding-seo/understanding-the-seo-dashboard-seo-recommendations.md)部分。

![](assets/image2014-9-17-21-3a32-3a22.png)

## 关键字排名摘要{#keyword-rank-summary}

此部分显示您对目标[关键字的网站排名。](/help/marketo/product-docs/additional-apps/seo/keywords/seo-add-keywords.md)

![](assets/image2014-9-17-21-3a34-3a5.png)

| 颜色 | 排名 |
|---|---|
| 绿色 | 搜索的第1个结果 |
| 黄色 | 第2-10个搜索结果 |
| 灰色 | 第11-20次搜索结果 |

>[!TIP]
>
>您还可以视图关键字的[特定列表](/help/marketo/product-docs/additional-apps/seo/keywords/seo-add-remove-keywords-from-a-list.md)的执行情况。 单击下拉列表，然后选择您感兴趣的对象。

滚！ 让我们继续走。

## 平均页面优化分数{#average-page-optimization-score}

“平均页面优化分数”是专有的Marketo分数，它以1到100的分数衡量您网站上的所有页面。

它基于数十种最佳做法的汇编。 得分为70或更高表示优化网站。

![](assets/image2014-9-17-21-3a35-3a55.png)

>[!TIP]
>
>您还可以查看特定页面列表的优化分数。 使用下拉列表，然后选择您感兴趣的选项。

## 关键字排名趋势{#keyword-rank-trends}

“关键字排名趋势”图表会以每月搜索量最高的五个卷来提取您的关键字，并绘制它们随时间的变化排名图。

>[!TIP]
>
>您可以通过单击从此图表中删除关键字。 试试看！ 图表将动态重建。

![](assets/image2014-9-17-21-3a37-3a1.png)
