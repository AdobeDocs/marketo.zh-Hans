# SEO — 手动添加入站链接URL {#seo-add-an-inbound-link-url-manually}

您可以手动添加入站链接。 下面介绍如何：

1. 转到&#x200B;**入站链接**&#x200B;部分。

   ![](assets/image2014-9-18-13-3a40-3a3.png)

1. 单击&#x200B;**添加URL**。

   ![](assets/image2014-9-18-13-3a40-3a8.png)

1. 键入URL。 单击&#x200B;**添加**。

   ![](assets/image2014-9-18-13-3a40-3a32.png)

   >[!TIP]
   >
   >您是否知道可以[将入站链接添加到新列表或现有链接](/help/marketo/product-docs/additional-apps/seo/understanding-seo/seo-managing-lists.md)? 看看！

   ![](assets/image2014-9-18-13-3a41-3a14.png)

滚！ 您现在应会看到新URL添加到列表。 收集所有数据可能需要一些时间。
