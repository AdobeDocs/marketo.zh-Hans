# SEO — 手动添加入站链接URL {#seo-add-an-inbound-link-url-manually}

您可以手动添加入站链接。 以下是操作方法：

1. 转到 **入站链接** 中。

   ![](assets/image2014-9-18-13-3a40-3a3.png)

1. 单击 **添加URL**.

   ![](assets/image2014-9-18-13-3a40-3a8.png)

1. 键入URL。 单击 **添加**.

   ![](assets/image2014-9-18-13-3a40-3a32.png)

   >[!TIP]
   >
   >你知道你能 [将您的入站链接添加到新列表或现有列表](/help/marketo/product-docs/additional-apps/seo/understanding-seo/seo-managing-lists.md){target=&quot;_blank&quot;}? 看看！

   ![](assets/image2014-9-18-13-3a41-3a14.png)

坚持！ 此时您应会看到已将新URL添加到列表。 收集所有数据可能需要一些时间。
