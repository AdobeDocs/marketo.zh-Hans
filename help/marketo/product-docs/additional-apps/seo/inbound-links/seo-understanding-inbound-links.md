# SEO — 了解入站链接{#seo-understanding-inbound-links}

入站链接指示搜索引擎您的网站值得引用。 真好！

![](assets/image2014-9-18-13-3a18-3a10.png)

## 列{#definition-of-columns}的定义

| 列标题 | 说明 |
|---|---|
| 入站链接的URL | 有关的网页。 |
| 优先级 | 此机会对您的页面排名有多重要。 |
| 指向竞争对手的链接 | 竞争对手是否在该URL中链接。 |
| 指向我的站点的链接 | 您的网站是否在该URL中链接。 |
| 来自我网站的链接 | 指示您的站点是否链接到该URL。 |
| PageRank | 标识搜索中的URL页面排名(1 - 10) |

真贴心！ 了解了入站链接后，我们可为您的网站建议更多入站链接机会。
