# SEO — 了解入站链接 {#seo-understanding-inbound-links}

入站链接指示搜索引擎，您的网站值得引用。 真好！

![](assets/image2014-9-18-13-3a18-3a10.png)

## 列的定义 {#definition-of-columns}

| 列标题 | 描述 |
|---|---|
| 入站链接的URL | 相关网页。 |
| 优先级 | 此机会对您的页面排名有多重要。 |
| 指向竞争者的链接 | 竞争者是否在该URL中链接到。 |
| 链接到我的网站 | 您的网站是否在该URL中链接到。 |
| 来自我网站的链接 | 指示您的网站是否链接到该URL。 |
| PageRank | 标识搜索中的URL页面排名(1 - 10) |

真贴心！ 现在，您已了解入站链接，因此我们可以为您的网站提供更多入站链接机会建议。
