# 视图推送通知仪表板{#view-the-push-notification-dashboard}

您可以轻松查看推送通知的效果。

1. 转至&#x200B;**营销活动**&#x200B;区域。

   ![](assets/image2015-12-11-12-3a57-3a48.png)

1. 选择活动。

   ![](assets/image2015-12-11-13-3a1-3a56.png)

1. 单击&#x200B;**视图:摘要**&#x200B;并选择&#x200B;**仪表板**。

   ![](assets/image2015-12-11-13-3a4-3a23.png)

1. 您可以以圆形图形视图iOS和Android的“已发送总数”和“总点击数”。 向下滚动以查看条形图中的点按衰减。

   ![](assets/image2015-12-15-15-3a23-3a47.png)

   >[!NOTE]
   >
   >_Sent_&#x200B;量度可反映的发送数量可能超过推送通知所发送给的确切人数。 这是因为它是根据有资格接收您的推送的&#x200B;*设备数*&#x200B;计算的。 例如，如果一个人有三个设备，则仪表板注册三个发送，而不是一个。
