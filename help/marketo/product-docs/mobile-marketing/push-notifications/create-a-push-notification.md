# 创建推送通知 {#create-a-push-notification}

创建推送通知很容易。 但是，在开始之前，您必须先让Marketo管理员和移动设备应用程序开发人员设置一些您在执行操作时需要的功能。 请参阅 [了解推送通知](/help/marketo/product-docs/mobile-marketing/push-notifications/understanding-push-notifications.md) 以了解详细信息。

1. 转到 **营销活动** 的上界。

   ![](assets/image2015-4-22-18-3a46-3a14.png)

1. 查找并选择您的项目。

   ![](assets/image2015-4-23-13-3a31-3a43.png)

1. 在 **新建**，单击 **新建本地资产**.

   ![](assets/image2015-4-23-13-3a33-3a20.png)

1. 选择 **推送通知**.

   ![](assets/image2015-4-23-13-3a35-3a6.png)

1. 输入 **推送通知名称** 单击 **创建**.

   ![](assets/image2015-4-23-13-3a36-3a56.png)

   真贴心！ 现在，创建推送通知后，让我们继续 [把它打扮成这样](/help/marketo/product-docs/mobile-marketing/push-notifications/configure-mobile-push-notification.md).
