# 创建推送通知{#create-a-push-notification}

很容易创建推送通知。 但是，在开始之前，您必须让Marketo Admin和Mobile Apps Developer设置您在进行操作时需要的一些内容。 有关详细信息，请参阅[了解推送通知](understanding-push-notifications.md)。

1. 转至&#x200B;**营销活动**&#x200B;区域。

   ![](assets/image2015-4-22-18-3a46-3a14.png)

1. 查找并选择您的项目。

   ![](assets/image2015-4-23-13-3a31-3a43.png)

1. 在&#x200B;**新建**&#x200B;下，单击&#x200B;**新建本地资产**。

   ![](assets/image2015-4-23-13-3a33-3a20.png)

1. 选择&#x200B;**推送通知**。

   ![](assets/image2015-4-23-13-3a35-3a6.png)

1. 输入&#x200B;**推送通知名称**&#x200B;并单击&#x200B;**创建**。

   ![](assets/image2015-4-23-13-3a36-3a56.png)

   真贴心！ 现在创建了推送通知，让我们继续[将其装好](configure-mobile-push-notification.md)。
