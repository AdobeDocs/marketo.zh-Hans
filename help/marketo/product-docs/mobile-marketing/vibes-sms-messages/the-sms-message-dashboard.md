# 短信消息仪表板 {#the-sms-message-dashboard}

短信消息仪表板显示有关您的短信消息的有用信息。

![](assets/converted-dashboard-image.png)

的 **短信进展** 图表显示发送总数和发送总数。 金额位于右侧，如果将鼠标悬停在条形图上，则会显示百分比。

![](assets/sms-progression-hand-border.png)

的 **概要** 图表以百分比形式显示计算的跳出率。 将鼠标悬停在拱形栏上可查看按金额和百分比划分的提交率。 将鼠标悬停在条形图的橙色“跳出率”部分上，可查看“软跳出率”和“硬跳出率”金额和百分比。

![](assets/hover-over-summary-hands-thin-border.png)

的 **一段时间内的活动** 图表允许您选择已发送总数或已交付总数。 从日期范围选择器中选择适当的范围。

![](assets/activity-over-time-hands.png)
