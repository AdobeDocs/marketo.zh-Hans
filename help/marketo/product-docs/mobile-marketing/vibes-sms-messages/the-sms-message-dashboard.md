# 短信消息仪表板 {#the-sms-message-dashboard}

短信消息仪表板显示有关短信消息的有用信息。

![](assets/converted-dashboard-image.png)

此 **SMS进度** 图形显示发送的总数和投放的总数。 金额位于右侧，如果将鼠标悬停在条形图上，则会显示百分比。

![](assets/sms-progression-hand-border.png)

此 **摘要** 该图以百分比显示计算出的跳出率。 将鼠标悬停在存档栏上，以查看按金额和百分比列出的交货率。 将鼠标悬停在栏的“橙色跳出率”部分上，可查看“软跳出率”和“硬跳出率”金额和百分比。

![](assets/hover-over-summary-hands-thin-border.png)

此 **一段时间内的活动** 图表允许您选择“已发送总计”或“已发送总计”。 从日期范围选择器中选择适当的范围。

![](assets/activity-over-time-hands.png)

按铃就清了！
