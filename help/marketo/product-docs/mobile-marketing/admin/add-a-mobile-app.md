# 添加移动应用程序{#add-a-mobile-app}

通过将您的移动App与Marketo连接，将推送通知发送给您的客户群。

应用程序通常在沙箱环境中开始，在沙箱中执行初始开发和测试。 然后，开发人员使用生产环境构建客户将使用的最终应用程序。 添加移动应用程序时，必须选择相应的通知证书（请参阅下面的步骤4）。

>[!AVAILABILITY]
>
>
>并非所有客户都购买了此功能。 有关详细信息，请与您的销售代表联系。

1. 单击&#x200B;**管理员**。

   ![](assets/image2015-4-22-16-3a12-3a32.png)

1. 单击&#x200B;**移动应用程序和设备**。

   ![](assets/image2016-1-12-15-3a42-3a30.png)

1. 单击&#x200B;**新建移动应用程序**。

   ![](assets/image2015-4-22-16-3a17-3a15.png)

1. 输入名称。 在&#x200B;**类型**&#x200B;下拉列表中，选择您使用的环境类型 — 沙箱或生产。 单击&#x200B;**保存**。

   ![](assets/image2015-11-18-15-3a52-3a15.png)

   >[!NOTE]
   >
   >我们建议您在生产环境中使用生产通知证书。 沙箱证书将安装在生产环境中，无问题，但您不会收到通知。 如果您对您的环境或通知证书有任何疑问，请与您的Marketo管理员或移动应用程序开发人员联系。

   不错！ 现在，让我们将您的应用程序配置为与Android和iOS设备一起使用。
