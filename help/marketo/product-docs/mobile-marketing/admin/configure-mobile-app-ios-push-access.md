# 配置移动应用程序iOS推送访问 {#configure-mobile-app-ios-push-access}

1. 单击&#x200B;**管理员**。

   ![](assets/image2015-4-22-16-3a12-3a32.png)

1. 选择&#x200B;**移动设备应用程序**。

   ![](assets/image2015-4-22-16-3a14-3a29.png)

1. 选择所需的移动设备应用程序。

   ![](assets/image2015-4-22-16-3a33-3a19.png)

1. 在“推送访问类型”下，选择iOS并单击&#x200B;**配置**。

   ![](assets/image2016-6-10-11-3a37-3a9.png)

   >[!NOTE]
   >
   >您需要移动应用开发人员提供的&#x200B;**证书**&#x200B;和&#x200B;**密码**。 开发人员可通过以下方式接收这些通知：登录Apple开发人员成员中心，为您的应用程序设置和下载推送通知证书，以及导出内容。 开发人员在导出时设置密码。 **重要信息**：证书应该适合您使用的环境类型 — 沙盒或生产。 请与您的Marketo管理员或移动应用程序开发人员确认。

1. 选择您的证书，输入密码，然后单击&#x200B;**保存**。

   ![](assets/image2015-4-22-17-3a19-3a18.png)

做得好！ 确保还使用Android配置应用程序。
