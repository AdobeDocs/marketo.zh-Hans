# 下载电子邮件的HTML {#download-an-emails-html}

出于备份和其他目的，Marketo允许您下载电子邮件的HTML内容。

1. 找到电子邮件并将其选中。 在&#x200B;**电子邮件操作**&#x200B;下拉列表中，单击&#x200B;**下载HTML**。

   ![](assets/one-4.png)

就这么简单！
