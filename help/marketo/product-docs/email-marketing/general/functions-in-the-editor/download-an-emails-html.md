# 下载电子邮件的HTML {#download-an-emails-html}

出于备份和其他目的，您可以使用Marketo下载电子邮件的HTML内容。

1. 找到电子邮件并将其选定。 在 **电子邮件操作** 下拉式单击 **下载HTML**.

   ![](assets/one-4.png)

这太简单了！
