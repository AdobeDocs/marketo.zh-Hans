# 创建日程表事件（.ics）文件 {#create-a-calendar-event-ics-file}

日历文件令牌允许您向Marketo电子邮件和登陆页添加日历事件(.ics)链接。

1. 在程序内，转到&#x200B;**[!UICONTROL My Tokens]**&#x200B;选项卡。

   ![](assets/image2014-9-11-15-3a33-3a27.png)

1. 将&#x200B;**[!UICONTROL Calendar File]**&#x200B;令牌拖到画布上。

   ![](assets/image2014-9-11-15-3a34-3a0.png)

1. 输入&#x200B;**令牌名称**&#x200B;并单击&#x200B;**[!UICONTROL Click to Edit]**。

   ![](assets/image2014-9-11-15-3a34-3a10.png)

1. 输入详细信息，然后单击&#x200B;**[!UICONTROL Save]**。

   ![](assets/image2014-9-11-15-3a34-3a16.png)

任务完成了！ 一定要测试一下。
