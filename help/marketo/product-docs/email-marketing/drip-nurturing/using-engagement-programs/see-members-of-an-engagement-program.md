# 请参阅参与项目的成员 {#see-members-of-an-engagement-program}

您可以快速查看参与计划的成员。 这是方法。

1. 在您的参与计划中，右键单击并选择 **查看成员**.

   ![](assets/membersofengagement.jpg)

   >[!NOTE]
   >
   >的  [内容已用尽](/help/marketo/product-docs/email-marketing/drip-nurturing/creating-an-engagement-program/understanding-engagement-programs.md) 列值在每次转换后都会更新。

1. 现在，您可以看到项目的所有成员。

   ![](assets/image2014-9-15-17-3a17-3a26.png)
