# 请参阅参与项目{#see-members-of-an-engagement-program}的成员

您可以快速查看参与项目的成员。 这是方法。

1. 在参与项目中，右键单击并选择&#x200B;**视图成员**。

   ![](assets/membersofengagement.jpg)

   >[!NOTE]
   >
   >每次转换后，将更新[已用尽的内容](/help/marketo/product-docs/email-marketing/drip-nurturing/creating-an-engagement-program/understanding-engagement-programs.md)列值。

1. 您现在可以看到项目的所有成员。

   ![](assets/image2014-9-15-17-3a17-3a26.png)
