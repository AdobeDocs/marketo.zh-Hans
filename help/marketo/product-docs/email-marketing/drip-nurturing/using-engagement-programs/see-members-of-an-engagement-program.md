# 查看参与计划的成员 {#see-members-of-an-engagement-program}

您可以快速查看参与计划的成员。 操作方法如下：

1. 从参与计划中，右键单击并选择&#x200B;**[!UICONTROL View Members]**。

   ![](assets/membersofengagement.jpg)

   >[!NOTE]
   >
   >每次转换后都会更新[已用完的内容](/help/marketo/product-docs/email-marketing/drip-nurturing/creating-an-engagement-program/understanding-engagement-programs.md)列值。

1. 您现在可以看到项目的所有成员。

   ![](assets/image2014-9-15-17-3a17-3a26.png)
