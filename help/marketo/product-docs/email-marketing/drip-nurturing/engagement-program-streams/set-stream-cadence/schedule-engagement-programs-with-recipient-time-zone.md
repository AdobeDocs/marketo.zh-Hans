# 按收件人时区计划参与计划 {#schedule-engagement-programs-with-recipient-time-zone}

当您计划参与项目流并且收件人时区处于活动状态时，节目群组将在第一个时区的午夜(UTC +14:00)开始运行。 我们要求您安排第一次演出 **至少25小时** 因为将来可能会有人在全球每个时区都有资格参加演员。 此时开始处理时，如果采用第一个时区，我们将确保在每个收件人的计划日期和时间发送电子邮件。

1. 在您的参与计划中，导航到 **流** 选项卡上，单击流的终止时间计划以对其进行编辑。

   ![](assets/image2017-12-5-13-3a36-3a21.png)

1. [设置频度设置](/help/marketo/product-docs/email-marketing/drip-nurturing/engagement-program-streams/set-stream-cadence.md) 照常，然后检查 **收件人时区** 框中。 请记住，您的首演必须在未来至少25小时内进行。 单击 **保存**.

   ![](assets/image2017-12-5-13-3a50-3a32.png)

1. 请注意，如果“收件人时区”处于活动状态，则终止时间计划将不显示特定时区，因为可能存在多个时区。 它将仅显示该小时。

   ![](assets/image2017-12-5-13-3a56-3a21.png)
