# 计划项目与收件人时区{#schedule-engagement-programs-with-recipient-time-zone}

当您计划参与项目流且收件人时区处于活动状态时，项目播放将在第一个时区的午夜(UTC +14:00)运行。 我们要求您在将来至少计划第一次演出&#x200B;**25小时**，因为可能有人有资格在全球每个时区进行该演出。 在第一个时区开始处理时，我们将确保在每个收件人的预定日期和时间发送电子邮件。

1. 在参与项目中，导航到&#x200B;**流**&#x200B;选项卡，然后单击流的节奏计划以编辑它。

   ![](assets/image2017-12-5-13-3a36-3a21.png)

1. [按照正常](/help/marketo/product-docs/email-marketing/drip-nurturing/engagement-program-streams/set-stream-cadence.md) 情况设置节奏设置，然后选中“收件人 **时间区** 域”框。请记住，您的首场演出将至少需要25小时。 单击&#x200B;**保存**。

   ![](assets/image2017-12-5-13-3a50-3a32.png)

1. 请注意，在收件人时区处于活动状态时，节奏计划将不显示特定时区，因为可能有多个时区。 它只显示小时。

   ![](assets/image2017-12-5-13-3a56-3a21.png)
