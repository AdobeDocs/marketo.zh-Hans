# 克隆流 {#clone-a-stream}

出于各种原因克隆流，包括测试不同的顺序和不同的节奏。

1. 选择您的参与计划，然后转到 **流** 选项卡。

   ![](assets/cloneasteam.jpg)

1. 单击流齿轮图标，然后单击 **克隆**.

   ![](assets/image2014-9-15-17-3a0-3a23.png)

   >[!NOTE]
   >
   >每个参与计划最多可以有25个流。

   做得好！ 你不喜欢让生活更轻松的东西吗？

   ![](assets/image2014-9-15-17-3a1-3a20.png)

   >[!CAUTION]
   >
   >流中的所有内容均被克隆（Cadence除外）。 请记住设置它。
