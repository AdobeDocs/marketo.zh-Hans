# 添加流 {#add-a-stream}

参与项目可以容纳多个流。 以下是如何添加新受众。

1. 转到 **营销活动**.

   ![](assets/login-marketing-activities-2.png)

1. 选择您的参与计划，然后单击 **流** 选项卡。

   ![](assets/streamstablifecycle.jpg)

1. 单击 **添加流**.

   ![](assets/image2014-9-15-16-3a56-3a23.png)

   >[!NOTE]
   >
   >每个参与计划最多可添加25个流。

   完成！

   ![](assets/image2014-9-15-16-3a56-3a27.png)
