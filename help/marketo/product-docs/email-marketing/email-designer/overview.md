# 电子邮件设计器概述 {#overview}

欢迎使用全新的 Adobe Marketo Engage 电子邮件设计器。

电子邮件设计器是 Marketo Engage 的最新创新，旨在通过提供可视化的拖放式编辑器以及开箱即用的标准模板，全面革新电子邮件及电子邮件模板的创建体验，从而提升工作效率和生产力。无需额外聘请第三方供应商，即可轻松创建自定义电子邮件模板。

## 如何访问  {#how-to-access}

+++了解如何访问电子邮件设计器。

要访问全新的电子邮件设计器，您的 Marketo Engage 订阅必须已迁移至 [Adobe Identity Management System (IMS)](https://experienceleague.adobe.com/zh-hans/docs/marketo/using/product-docs/administration/marketo-with-adobe-identity/adobe-identity-management-overview)。如果您的订阅尚未完成迁移，且希望加快进度，请联系 Adobe 客户团队（您的客户经理）或 [Marketo 支持团队](https://nation.marketo.com/t5/support/ct-p/Support)。

### 添加用户 {#add-users}

1. 在 Marketo Engage 中，前往 **[!UICONTROL Admin]** 区域并选择 **[!UICONTROL New Email Designer]**。

   ![在管理员左侧导航栏中选择新电子邮件设计器](assets/overview-1.png){width="600" zoomable="yes"}

1. 单击 **[!UICONTROL Add Users]**。

   ![“添加用户”按钮](assets/overview-2.png){width="600" zoomable="yes"}

1. 在 **[!UICONTROL Add New Values]** 下拉列表中，选择所需的用户。完成后，单击 **[!UICONTROL OK]**。

   ![从列表中选择用户](assets/overview-3.png){width="600" zoomable="yes"}

+++

## 常见问题解答 {#faq}

**旧版电子邮件编辑器是否会继续可用？如果会，可用到什么时候？**

是的，旧版（经典）电子邮件编辑器将与新的电子邮件设计器并行存在。在经典编辑器中创建的电子邮件将继续保留。目前尚未确定经典编辑器的弃用时间，但它将至少持续可用至 2026 年。在弃用旧版编辑器之前，我们将提前数月发出通知。

**哪些项目可以使用新电子邮件设计器创建的电子邮件？**

新电子邮件设计器创建的电子邮件可在所有项目中使用（唯一的例外是交互式网络研讨会项目）。同时也支持克隆。

**现有的电子邮件模板能否在新设计器中使用？**

可以，但[需要先进行导入](/help/marketo/product-docs/email-marketing/email-designer/import-template.md)。

**是否可以轻松将新设计器中的资源移动到其他项目？**

可以！

**是否可以通过 API 编辑新电子邮件设计器中的资源？**

目前，新电子邮件设计器尚不支持通过 API 编辑资源。

**是否可以强制统一品牌规范（字体、徽标和颜色）？**

可以！通过[品牌主题](/help/marketo/product-docs/email-marketing/email-designer/brand-themes.md)创建和管理您的品牌规范。

**模板模块的创建是否更具可定制性和可扩展性？**

是的，完全可以。借助我们的 WYSIWYG（所见即所得）编辑器，模板模块将更具可定制性。

**新设计器中的电子邮件模板创建流程是怎样的？是所见即所得，还是需要 HTML 知识？**

这是一个所见即所得的过程，不需要 HTML 知识。您可以在设计器中轻松创建模板，从而减少对外部 Web 开发人员的依赖。不过，您仍然可以选择通过 HTML 更新 CSS 或编辑单个区块。

**新电子邮件设计器是否支持 AMP 语言？**

目前尚不支持 AMP。

**如何确认我们的 Marketo Engage 订阅是否已迁移至 IMS (Adobe Admin Console)？**

如果您通过 [Adobe Experience Cloud](https://experiencecloud.adobe.com/){target="_blank"} 登录 Marketo Engage，则表示您的订阅已完成迁移。

**新电子邮件设计器支持哪些浏览器？**

目前建议使用 Google Chrome、Apple Safari、Microsoft Edge 或 Mozilla Firefox。
