# Web区段 {#web-segments}

## 查看区段 {#view-segment}

![](assets/image2014-11-11-20-3a24-3a5.png)

区段选项卡显示您根据各种属性设置的所有自定义区段。  **区段是符合“设置区段”页面中定义的指定条件的访客集合。**  区段可以是来自特定行业、位置或基于访客现场活动的访客。

在Web个性化中，访客可以匹配多个区段。 例如，如果存在适用于美国访客的区段和适用于金融公司的区段，则来自美国银行的Web访客将匹配 **两者** 为美国访客之分部及为金融公司之分部。

**区段 — 右侧面板**

![](assets/image2014-11-12-10-3a46-3a32.png)

在表格中选择区段会在右侧面板中显示有关该区段的更多详细信息。

这些详细信息包括：

* 区段名称
* 区段的创建日期
* 显示与该区段一起运行的促销活动的关联促销活动。 单击反应数量将会转到营销活动页面，该页面显示区段的营销活动CTA（行动号召）
* 区段的匹配数量（符合区段标准的访客数量）和匹配区段的独特（独特）访客数量。 单击独特访客链接将会转到显示区段结果的访客页面
* 区段的所有者/用户创建者
* 与区段关联的域站点
* 区段选定标准的简短摘要

## 启用或禁用区段 {#enable-or-disable-a-segment}

![](assets/image2014-11-12-10-3a48-3a9.png)

要启用或禁用区段，请选中表中该区段的复选框，然后在表底部的“选择操作”下拉框中，选择操作“启用”或“禁用”。 禁用区段后，“状态”列下将显示“禁用”一词。

## 创建区段 {#create-segments}

您创建的区段符合您在 **设置区段** 页面。 您也可以根据一系列标准来自定义区段，并在营销活动中定位特定受众。

创建新区段

从 **区段** 页面，单击 **新建** 在图表下。 出现以下屏幕。

![](assets/four.png)

定义区段的常规参数：

* **名称：**  命名您的区段。
* **描述：**  提供对区段标准的更详细说明。
* **域：**  选择要包含在区段中的域。
* **区段规则逻辑：**  选择AND / OR逻辑以构建每个分段属性
* **计时：** 定义您希望在营销活动中参与的访客参与度级别

   * **进入时**：从访问网站的访客开始参与
   * **第1 - 9次单击后**：在网站上的特定点击量后吸引访客

>[!TIP]
>
>**区段规则逻辑**
>
>有三个筛选选项：
>
>1. 使用所有筛选器（1和2及3...）
>1. 使用任意过滤器（1或2或3...）
>1. 高级过滤器（使用和/或表达式）
>
>    高级过滤器允许您控制区段条件。 输入以“and”和“or”分隔的过滤器编号。
>
>    * 1和2及3
>    * 1个、2个或3个
>
>    将“and”和“or”混合在一起需要用括号来阐明逻辑意图。 例如，“1或2和3”必须写成以下内容之一：
>
>    * 1和（2或3）
>    * （1和2）或3
>
>    对于更复杂的逻辑(例如，
>
>    * （1和2）或（3和4）
>    * 1和(2或（3和4）)
>
>    在插入、删除或重新排序任何内容后，检查您的逻辑。

将区段属性从右侧列拖放到左侧的区段编辑器中：

![](assets/five.png)

### Firmographics {#firmographics}

**位置**

拖放 **位置** 到区段编辑器中。

* 从以下参数中选择：

   * **包括**  — 选择您希望营销活动包含还是排除位置。
   * **选择要添加的国家/地区**  — 从下拉框中，选择要包含在区段中的国家/地区。 国家/地区名称显示在右侧。 您可以选择多个国家/地区。

添加国家/地区后，您还可以指定区段的州/省、城市和邮政编码。

* **选择要添加的省/市/自治区**  — 从下拉框中，选择要包括的美国州或加拿大省。 您可以进行多个选择。
* **邮政编码**  — 输入要包含在区段中的邮政编码。
* **城市**  — 输入希望包含的一个或多个城市。 在城市之间使用分号。

>[!TIP]
>
>**我应选择哪些区段条件？ “AND”或“OR”？** OR在每个字段中用作附加选项。 潜在客户只需满足在每个字段内选择的多个标准中的一个标准，即可获得该区段的资格。 (例如，潜在客户可能来自美国。 *或* 来自国防工业的)。 AND函数作为此区段必须满足的附加必需参数。 （例如，潜在客户必须既来自美国，也来自国防工业）。 在每个分段配置文件中，每个单独的字段都可以同时用作两个字段，可以是“AND”或“OR”，具体取决于所选的区段条件。

**行业** 在 **配置文件分段** 部分，选中旁边的框 **行业**.

* 从以下参数中选择：

   * **包括**  — 选择您希望区段包括还是排除某个行业。
   * **选择要添加的行业**  — 选择要包含在区段中的行业。 行业出现在下拉框的下方。 您可以选择多个行业。

**组织组**

在 **配置文件分段** 部分，选中旁边的框 **组织组。**

* 从下拉框中，从以下选项中选择：

   * 《财富500强》 — 仅包括《财富500强》公司
   * 《财富1000强》 — 仅包括《财富1000强》公司在此部分
   * Global 2000 — 在此部分包括Global 2000公司
   * 企业 — 包括员工超过1,000人且收入超过2.5亿美元的组织
   * SMB — 仅包括此部门中的中小型企业

**组织**

* **来自这些公司（特定名称）**

   * 从“选择要添加的公司”下拉列表中选择要定位的公司。
   * 您可以键入要定位的确切组织名称。 *它是 _始终_ 建议使用指定帐户列表，而不是手动键入名称以更好地匹配（请参阅下文）。

**指定帐户列表**

从中选择 [指定帐户列表](/help/marketo/product-docs/web-personalization/account-based-web-marketing/create-a-new-account-list.md) 以细分关键目标帐户。

### 已知人员 {#known-people}

**数据库**

Web个性化与您的Marketo数据库集成，允许您按已知人员属性和数据对营销活动进行细分和个性化。

选择数据库，然后从下拉列表中选择一个人员数据字段。 选择 **+** 以从下拉列表中添加字段。

![](assets/seven.png)

您可以在“帐户设置”>“数据库”中添加或删除人员数据字段

>[!TIP]
>
>根据Marketo人员的所有人员数据字段（如职务、分数、角色等）创建区段标准。
>
>例如： “职位等于CMO”和“得分小于或等于50”

**Marketo电子邮件营销活动** 通过访客点击Marketo电子邮件并到达网站，通过电子邮件反向链接对营销活动进行分段和个性化。 按Marketo项目名称或营销活动名称进行分段，并继续通过电子邮件到Web的对话。 选择+以从下拉列表中添加字段。

![](assets/image2015-5-27-17-3a20-3a34.png)

**状态**

根据潜在客户的状态定义区段：已知或匿名。

* 已知 — 从已知访客的下拉框中选择此选项。 当访客在您的网站上提交表单并出现在Web个性化的“人员”页面中时，就知道访客。
* 匿名 — 从匿名访客的下拉框中选择此选项。

![](assets/image2015-5-27-17-3a23-3a2.png)

### 行为 {#behavioral}

**访问次数 —** 根据访客行为或标识定义区段。

* 访问次数 — 从下拉框中选择此选项，可指定潜在客户对网站的访问次数。

   * 从下拉框中选择等于、等于或大于，或者等于或小于。

* 特定访问 — 从下拉框中选择此选项可指定特定访客。

   * 在右侧的文本框中，输入要跟踪的访客编号。 单击访客（在访客页面中）时，可以在右侧面板的Set Campaign中找到唯一的Web个性化访客标识号。 访客ID位于高级设置部分。 访客ID也可以在URL中找到（例如VISITOR=JZZJIFJNUI60PZ8Y97BHTY9BL8PKWS）。

**搜索词**  — 根据潜在客户的搜索词定义区段。

* 搜索的访客 — 从下拉列表中，从访客搜索中选择要跟踪的搜索词，或添加您自己的搜索词。 (不需要 &#42; 搜索词的通配符（设置为默认包含包含搜索词的短语）。

**反向链接**  — 添加访客引用的URL。

* 选择要添加的反向链接 — 从下拉列表中，选择要跟踪的反向链接网站或添加您自己的反向链接。 选中后，反向链接将显示在下面的框中。 (使用 &#42; （因为允许使用通配符）

**包含页面**  — 跟踪潜在客户在您网站上访问的特定页面。

* URL匹配 — 添加要跟踪的特定网页的URL。 您可以通过用分号分隔多个URL来添加这些URL。 (使用 &#42; （允许使用通配符）。

**排除页面**  — 排除您不希望在区段中匹配的特定页面。 (使用 &#42; （允许使用通配符）。

* URL不匹配 — 添加要从跟踪中排除的特定网页的URL。 您可以通过用分号分隔多个URL来添加它们

## 编辑区段 {#edit-segments}

您可以编辑已创建的区段。

1. 要编辑区段，请转到 **区段**.

   ![](assets/image2014-11-12-11-3a38-3a22.png)

1. 在 **区段** 表格中，单击编辑图标( ![](assets/segment-edit.png))。 此 **设置区段** 此时将打开页面，并显示选定的区段。
1. 应用您想要对区段所做的任何编辑或更改。
1. 单击 **保存**.

## 删除区段 {#delete-segments}

您可以删除已创建的区段。

1. 从 **区段** 选择区段。
1. 单击删除图标( ![](assets/segment-delete.png) )。
1. 此时将显示一条确认消息，确认您将删除 **区段**.

>[!NOTE]
>
>您无法删除与营销活动关联的区段。 首先，您需要删除营销策划，然后删除区段。
