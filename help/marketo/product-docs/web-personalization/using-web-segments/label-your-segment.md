# 为区段添加标签 {#label-your-segment}

您是否拥有如此多的区段，以至于滚动会变得十分繁琐？ 可使用标签标记区段，以便快速查找。

## 标记区段 {#tag-a-segment}

1. 登录[!DNL Web Personalization]并转到&#x200B;**[!UICONTROL Segments]**。

   ![](assets/new-dropdown-segments-hand.jpg)

1. 选择要使用标签标记的区段。

   ![](assets/image2015-10-14-15-3a26-3a28.png)

1. 要使用现有标签，请单击&#x200B;**[!UICONTROL Set Label]**，选中一个框，然后单击&#x200B;**[!UICONTROL Apply]**。

   ![](assets/image2015-10-14-15-3a34-3a42.png)

1. 或者，要创建新标签，请单击&#x200B;**[!UICONTROL Set Label]**，输入新标签名称，然后单击&#x200B;**新建**。

   ![](assets/image2015-10-14-15-3a38-3a30.png)

   >[!NOTE]
   >
   >“新建”按钮显示新标签的名称。 如果标签太长，则可能不会显示“新建”。

酷！ 现在，您已了解如何为区段分配和创建标签。
