# 编辑区域{#edit-regions}

是否希望更改用户区域设置以仅查看特定区域的数据？

1. 转到&#x200B;**用户设置**。

   ![](assets/image2014-12-1-23-3a8-3a40.png)

1. 单击&#x200B;**编辑区域**。

   ![](assets/image2014-12-3-18-3a55-3a25.png)

1. 检查您所在的国家或地区或与您所在地区相关的州。

>[!NOTE]
>
>选择“美国”后，将在页面底部打开所有要选择的“美国”选项。
