# 编辑区域 {#edit-regions}

想要将用户区域设置更改为仅查看特定区域的数据吗？

1. 转到&#x200B;**用户设置**。

   ![](assets/image2014-12-1-23-3a8-3a40.png)

1. 单击&#x200B;**编辑区域**。

   ![](assets/image2014-12-3-18-3a55-3a25.png)

1. 检查与您的地区相关的国家或州。

>[!NOTE]
>
>选择美国，将在页面底部打开所有美国选项以供选择。
