# 了解内容分析 {#understanding-content-analytics}

“内容分析”页面显示您从您的网站中发现的现有内容（案例研究、博客文章、视频、新闻稿等）。 它还显示您的内容以及访客与其互动时生成的人员的性能。

## 视图内容分析 {#view-content-analytics}

转到内 **容分析**。

![](assets/one.png)

在“内容分析”页面上，您可以：

* 按时间范围（日、周和月）过滤
* 按内容标题和内容URL搜索
* 单击视图、直接转换和转化率的列标题，按降序或升序排序。

![](assets/content-analytics-8-29-16-blur.png)

您还可以通过单击该图标以CSV格式导出文件。

![](assets/image2016-8-29-13-3a51-3a49.png)

## 视图附加内容分析 {#view-additional-content-analytics}

单击内容片段的分析图标。

![](assets/four.png)

此时将打开一个对话框，其中包含该特定内容片段的其他内容分析。

![](assets/five.png)

其他内容分析包括：

**分析**

* **视图**:所选时间范围的视图内容片段
* **直接转换**:在同一次访问中查看内容并填写表单的Web访客。
* **转化率****:**由直接转化除以点击量计算的转化率百分比

**趋势**

* 一个 **趋势图** ，显示特定内容片段的最近30天视图。 将指针悬停在线形图上，可查看特定日期的内容视图数

## 删除内容 {#delete-content}

![](assets/six.png)

在“内容分析”页面中，单击要删除的内容的删除图标。 此时会显示一条消息，确认您要删除内容。
