# Web个性化概述 {#web-personalization-overview}

## 简介 {#introduction}

对于新的Marketo客户，实时个性化分为四个应用程序，使用以下两个拼贴进行访问：Web个性化和内容`<sup>AI</sup>`。

![](assets/pasted-image-at-2016-03-23-02-45-pm.png) ![](assets/mlm-homepage-content-ai-281-29.png)

>[!NOTE]
>
>Web个性化和内容`<sup>AI</sup>` ，提供所有市场支持的语言（英语、法语、德语、日语、葡萄牙语、西班牙语）。

现有客户在续订之前，会继续通过实时个性化拼贴访问应用程序，届时他们将使用上述登录名。

![](assets/image2016-2-9-8-3a52-3a32.png)

单击“Web个性化”拼贴时，将显示仪表板页。 单击“内容`<sup>AI</sup>` ”拼贴，然后直接转到“内容”“Recommendations”页面。

这四个应用是：

* Web个性化
* 基于帐户的Web营销
* 网站重定位
* 内容`<sup>AI</sup>`

** Web个性化拼 **贴是您创建个性化Web****活动的访问点。** **基于帐户的Web营销和Web重定向。 您可以在“内容分析”页面上添加内容。

**基于帐户的Web营销** ，可与Web个性化和指定帐户功能配合使用。 “指定帐户”页提供有关现有客户、总参与和组织趋势的图形信息，以及帐户列表。 您还可以在那里创建新的指定帐户。

**“网站重定** 位”使您能够针对细分受众进行再营销。 您可以设置区段，但不能在筛选器中使用指定帐户。

**内容`<sup>AI</sup>`** 是提供内容建议的地方。 您可以轻松地向“内容Recommendations”页面添加内容。

>[!NOTE]
>
>要访问基于帐户的Web营销或网站重定位，您必须单击“Web个性化”拼贴，即使您没有Web个性化应用程序也是如此。

## 组织 {#organizations}

“组织”选项卡显示在指定时间段内访问您网站的组织的所有详细信息(名称、位置、活动和时间戳)。 表可以按时间、位置、域以及自由文本搜索进行排序和组织。

![](assets/image2014-11-10-19-3a23-3a18.png)

>[!TIP]
>
>“最活跃”与“Latest”（最新）-表格由已识别的组织和Internet服务提供商（由ISP图标表示）组织。 它可以按照以下方式进行排序：
>
>* 最活跃：表中最活跃的组织，基于页面视图数
>* 最新：表中最近的组织（默认）
>* 要获得有价值的洞察，请根据最活跃的

>

**共享组织**

要将您的网站访客的信息共享给同事？ 单击“组织”，然后单击右侧面板上的“共享”链接。

此操作会激活“共享此组织”叠加，在该叠加中，您可以发送电子邮件，共享访问网站的组织的名称和详细信息。

![](assets/image2014-11-10-19-3a25-3a42.png)
