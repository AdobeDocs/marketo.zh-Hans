# Sales Intelligence {#sales-intelligence}

Marketo Web个性化提供了有关访问您网站的匿名和已知Web访客和组织的重要信息。 销售主管可以使用它来跟踪、监控和接收关键组织和帐户的警报，并根据具体地区提供警报。

通过登录Marketo Web个性化平台、通过每日/每周电子邮件报表，或在移动设备应用程序上查看，可访问该平台。

## 定义用户区域 {#define-the-users-region}

要根据您所在的地区、国家/地区或州，定义在Marketo Web个性化中看到的组织和访客数据，请转到 [用户设置](/help/marketo/product-docs/web-personalization/getting-started/user-settings.md) 选择国家或州。

## 设置每日或每周电子邮件报表 {#set-up-daily-or-weekly-email-reports}

在 [用户设置](/help/marketo/product-docs/web-personalization/getting-started/user-settings.md)，则可以设置为接收凌晨发送的每日或每周电子邮件报告。 销售主管感兴趣的主要报告包括“组织”和“潜在客户”报表，其中显示用户特定区域访问的排名最前的组织和提交的潜在客户，包括其行为。

>[!NOTE]
>
>我们建议您在Marketo Web Personalization中设置具有仅查看权限的销售主管。 这意味着他们可以查看所有页面，但不能创建、编辑或删除任何功能。
