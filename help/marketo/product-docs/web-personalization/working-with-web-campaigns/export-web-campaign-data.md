# 导出Web营销活动数据 {#export-web-campaign-data}

按照以下简单步骤导出Web促销活动数据。

1. 转到 **Web营销活动**.

   ![](assets/one-2.png)

1. 在页面的右上方，单击导出CSV图标。

   ![](assets/two-2.png)

1. 打开或保存文件。

   ![](assets/three-2.png)

1. 查看您的文件以查看有用的统计信息。

   ![](assets/four-1.png)
