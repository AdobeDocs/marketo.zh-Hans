# 导出Web活动数据{#export-web-campaign-data}

按照以下简单步骤导出Web活动数据。

1. 转至&#x200B;**Web活动**。

   ![](assets/one-2.png)

1. 在页面的右上方，单击“导出CSV”图标。

   ![](assets/two-2.png)

1. 打开或保存文件。

   ![](assets/three-2.png)

1. 视图您的文件，查看有用的统计信息。

   ![](assets/four-1.png)
