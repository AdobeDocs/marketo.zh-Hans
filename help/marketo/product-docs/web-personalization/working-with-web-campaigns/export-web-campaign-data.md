# 导出Web Campaign数据 {#export-web-campaign-data}

按照这些简单步骤导出Web促销活动数据。

1. 转到 **Web营销活动**.

   ![](assets/one-2.png)

1. 单击页面右上方的导出CSV图标。

   ![](assets/two-2.png)

1. 打开或保存文件。

   ![](assets/three-2.png)

1. 查看文件以查看有用的统计信息。

   ![](assets/four-1.png)
