# 编辑现有Web活动{#edit-an-existing-web-campaign}

1. 转到&#x200B;**Web活动**。

   ![](assets/image2016-8-18-16-3a15-3a14.png)

1. 在&#x200B;**Web活动**&#x200B;页面上，单击要编辑的活动上的&#x200B;**编辑**。

   ![](assets/web-campaigns-1-edit-hand.png)
