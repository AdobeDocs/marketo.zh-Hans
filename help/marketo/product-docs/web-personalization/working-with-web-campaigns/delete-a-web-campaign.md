# 删除Web活动{#delete-a-web-campaign}

1. 转至&#x200B;**Web活动**。

   ![](assets/web-campaigns-hand-3.jpg)

   >[!NOTE]
   >
   >要更轻松地找到所需的Web活动，请使用[过滤器功能](filter-web-campaigns.md)。

1. 在Web活动页上，单击要删除的活动上的**删除**。

   ![](assets/web-campaigns-1-delete-hand-1.png)

1. 将显示一条确认消息，确认是否要删除Web活动。
