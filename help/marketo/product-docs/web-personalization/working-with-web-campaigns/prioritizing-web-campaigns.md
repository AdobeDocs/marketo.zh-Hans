# 对Web促销活动进行优先级排序 {#prioritizing-web-campaigns}

通过设置优先级分数，在两个或多个Web营销活动重叠时，优先处理您的Web营销活动。

>[!NOTE]
>
>**重叠的营销活动**
>
>Web营销活动在以下情况下发生重叠：
>
>* 两个或多个小组件和/或对话框营销活动在同一页面上同时做出反应
>* 具有相同区域ID的两个或多个In Zones在同一网页上同时做出反应
>
>区域内营销活动和（小组件或对话框）营销活动可以在同一页面上做出反应。

1. 转到“Web营销活动”。

   ![](assets/web-campaigns-hand-6.jpg)

   >[!NOTE]
   >
   >为了更便于查找所需的Web营销活动，请使用 [过滤功能](/help/marketo/product-docs/web-personalization/working-with-web-campaigns/filter-web-campaigns.md).

1. 在“编辑营销活动”页面中，设置优先级得分（9999 =最高优先级1 =最低优先级）。

   ![](assets/image2015-7-9-20-3a20-3a58.png)

   >[!TIP]
   >
   >我们建议您仅在存在潜在的营销活动重叠且其中一个营销活动的重要性较高时，才使用营销活动优先级得分。 您不必为每个营销活动设置优先级。

1. 保存或启动营销活动。

1. 请参阅“网络营销活动”页面中显示的优先级得分。

![](assets/web-campaign-priority-score.jpg)
