# 存档Web活动 {#archive-a-web-campaign}

1. 转到 **Web活动**。

   ![](assets/one.jpg)

1. 单击所需Web活动的状态下拉框，然后选择“存 **档”**。

   ![](assets/two-3.png)

   >[!NOTE]
   >
   >存档的Web活动将不会显示在默认筛选器中。 要查看这些文件，请单击“筛选器”图标， **然后在**“状态”下 **选中“存** 档 **”复选框，然**&#x200B;后单击“应用”。
