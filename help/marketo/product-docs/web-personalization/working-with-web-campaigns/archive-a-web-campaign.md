# 存档Web活动 {#archive-a-web-campaign}

1. 转到 **Web营销活动**.

   ![](assets/one.jpg)

1. 单击所需Web营销活动的状态下拉列表，然后选择 **存档**.

   ![](assets/two-3.png)

   >[!NOTE]
   >
   >已存档的Web营销活动将不会显示在默认过滤器中。 要查看这些过滤器，请单击过滤器图标，然后单击 **状态**，选择 **已存档** 复选框，然后单击 **应用**.
