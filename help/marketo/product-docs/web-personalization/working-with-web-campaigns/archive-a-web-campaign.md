# 存档Web活动 {#archive-a-web-campaign}

1. 转到&#x200B;**网络营销活动**。

   ![](assets/one.jpg)

1. 单击所需Web营销活动的状态下拉列表，然后选择&#x200B;**存档**。

   ![](assets/two-3.png)

   >[!NOTE]
   >
   >已存档的Web营销活动将不会显示在默认过滤器中。 若要查看，请单击“筛选器”图标，在&#x200B;**状态**&#x200B;下，选中&#x200B;**已存档**&#x200B;复选框，然后单击&#x200B;**应用**。
