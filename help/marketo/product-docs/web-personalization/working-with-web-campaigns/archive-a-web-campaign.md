# 归档 Web 营销活动 {#archive-a-web-campaign}

1. 前往 **[!UICONTROL Web Campaigns]**。

   ![](assets/one.jpg)

1. 单击所需Web营销活动的状态下拉列表，然后选择&#x200B;**[!UICONTROL Archive]**。

   ![](assets/two-3.png)

   >[!NOTE]
   >
   >已存档的Web营销活动将不会显示在默认过滤器中。 若要查看这些报表，请单击“筛选器”图标，然后在&#x200B;**[!UICONTROL Status]**&#x200B;下选中&#x200B;**[!UICONTROL Archived]**&#x200B;复选框，然后单击&#x200B;**[!UICONTROL Apply]**。
