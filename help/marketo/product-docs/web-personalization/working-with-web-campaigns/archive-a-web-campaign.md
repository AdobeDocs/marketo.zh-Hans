# 存档Web活动{#archive-a-web-campaign}

1. 转到&#x200B;**Web活动**。

   ![](assets/one.jpg)

1. 单击所需Web活动的状态下拉框，然后选择&#x200B;**存档**。

   ![](assets/two-3.png)

   >[!NOTE]
   >
   >存档的Web活动将不显示在默认筛选器中。 要查看这些文件，请单击“过滤器”图标，并在&#x200B;**Status**&#x200B;下，选中&#x200B;**Archived**&#x200B;复选框，然后单击&#x200B;**Apply**。
