# 按最新或最佳表现排序 Web 营销活动 {#sort-web-campaigns-by-latest-or-top-performing}

您可以通过几种不同的方式对Web营销活动进行排序。

1. 前往 **[!UICONTROL Web Campaigns]**。

   ![](assets/web-campaigns-hand-1.jpg)

1. 选择排序类型。

   >[!NOTE]
   >
   >**定义**
   >
   >**[!UICONTROL Latest]** — 将根据营销活动的创建日期排序。 顶部的最新营销活动。
   >
   >**[!UICONTROL Top Performing]** — 将根据点进率对营销活动进行排序。 顶部的最高点进。

   ![](assets/image2016-11-4-13-3a34-3a59.png)

   是的，真的那么简单。
