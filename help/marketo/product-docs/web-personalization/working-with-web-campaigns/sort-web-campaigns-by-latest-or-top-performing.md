# 按最新或最佳活动排序{#sort-web-campaigns-by-latest-or-top-performing}

您可以用几种不同的方式对Web活动进行排序。

1. 转到&#x200B;**Web活动**。

   ![](assets/web-campaigns-hand-1.jpg)

1. 选择排序类型。

   >[!NOTE]
   >
   >**定义**
   >
   >**最新**  — 将根据创建活动的日期进行排序。最新活动。
   >
   >**表现最佳**  — 将根据点进率对活动进行排序。顶部的最高点击。

   ![](assets/image2016-11-4-13-3a34-3a59.png)

   是的，这真的很简单。
