# 使用Web个性化富文本编辑器 {#using-the-web-personalization-rich-text-editor}

![](assets/one.png)

“设置促销活动”页面中的Web个性化富文本编辑器允许进行文本格式设置、链接和图像插入。 它包括与标准字处理器类似的格式选项。

>Web令牌使用 [人员数据](/help/marketo/product-docs/web-personalization/using-web-segments/manage-person-data.md) 在Web个性化中管理的Marketo数据库中的帐户设置>数据库。 如果Web令牌是帐户设置>数据库中尚未添加的新数据库字段，则会自动添加该令牌，并且最长可能需要24小时才能激活。
