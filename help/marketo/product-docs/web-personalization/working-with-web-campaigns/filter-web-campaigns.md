# 过滤Web营销活动 {#filter-web-campaigns}

在您创建了数百个Web个性化促销活动后，使用过滤器仅查看您感兴趣的促销活动，这真的会有所帮助。

1. 转到 **Web营销活动**.

   ![](assets/web-campaigns-hand-8.jpg)

1. 在“Web营销活动”页面上，单击 **过滤器**.

   ![](assets/web-campaigns-page-filter-hand.jpg)

1. 选中要筛选的营销活动状态和/或类型复选框，例如， **已暂停** 或 **对话框**. 单击 **应用**.

   ![](assets/web-campaigns-filters-hands.jpg)

   >[!TIP]
   >
   >使用 **全选** 用于选择全部或 **清除** 链接以清除所有复选框。

1. 现在，只显示与您的过滤器匹配的营销活动。

   ![](assets/web-campaigns-filter-only-paused.jpg)

   一块蛋糕！
