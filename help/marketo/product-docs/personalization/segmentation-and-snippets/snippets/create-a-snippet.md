# 创建代码片段 {#create-a-snippet}

代码片段可用作 **电子邮件** 和 **登陆页面**.

1. 转到 **Design Studio。**

   ![](assets/designstudio.png)

1. 单击 **新建** 然后 **新代码片段**.

   ![](assets/image2014-9-16-8-50-4.png)

1. 输入所需的详细信息并单击 **创建**.

   ![](assets/image2014-9-16-8-3a50-3a14.png)

干得好！ 通过为动态内容创建片段来简化工作。 现在你可以继续 [将内容添加到新代码片段](/help/marketo/product-docs/personalization/segmentation-and-snippets/snippets/add-content-to-a-snippet.md).
