# 调查潜在客户 {#investigate-lead}

了解为什么有人没有MQL、没有计划资格或在生命周期中前进。

要查看调查方案的完整列表，请单击右侧的箭头。

屏幕快照

| **标题** | 描述 |
|----------|------------|
| **MQL调查** | 提供有关特定潜在客户不符合条件、停止参与或卡住的详细信息。 |
| **计划资格** | 提供有关Lead为何不符合培养计划的资格的详细信息。 |
| **生命周期进度** | 用于探索为什么商机停滞在比预期更长的阶段中。 |
| **参与分析** | 当Lead停止打开或单击电子邮件时使用。 |
| **促销活动历程** | 用于查看历程中商机的完整路径和接触点。 |
| **数据质量** | 当可能预期错误数据导致Lead出现问题时使用。 |
| **销售转移** | 在Sales Rejected a Lead且未跟进时使用。 |
| **归因** | 用于探索什么影响了Lead并促成了Opportunity。 |
| **得分衰减** | 调查Lead得分意外下降的原因。 |

>[!TIP]
>
>要快速查看所有九种方案的列表，请在顶部的提示窗口中键入`/lead investigation`。
