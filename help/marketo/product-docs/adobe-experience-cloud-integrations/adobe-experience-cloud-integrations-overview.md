# Adobe Experience Cloud集成概述{#adobe-experience-cloud-integrations-overview}

Adobe提供了一个强大的需求营销解决方案，首先是在Adobe Experience Cloud产品中同步本机受众。 在Adobe Audience Manager中利用以B2B为中心的独特数据，为现有Adobe应用程序释放强大的需求营销用例和未开发的潜力。

>[!NOTE]
>
>这些集成(包括Audience Manager和Adobe Real-Time CDP Marketo连接器)可能需要在Marketo Engage中映射IMS组织。 [了解如何设置Adobe组织映射](/help/marketo/product-docs/adobe-experience-cloud-integrations/set-up-adobe-organization-mapping.md){target=&quot;_blank&quot;}。
