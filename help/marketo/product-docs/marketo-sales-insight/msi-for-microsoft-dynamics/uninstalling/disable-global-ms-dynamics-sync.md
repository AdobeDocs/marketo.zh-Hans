# 禁用全局MS Dynamics同步 {#disable-global-ms-dynamics-sync}

按照以下简单步骤禁用MS Dynamics同步。

1. 在Marketo中，单击“管 **理员**”。

   ![](assets/one.png)

1. 在“集成”下，单 **击“Microsoft Dynamics**”。

   ![](assets/two.png)

1. 单击 **禁用同步**。

   ![](assets/three.png)

   >[!NOTE]
   >
   >如果实例中未显示“禁用同步”按钮，请与Marketto [支持联系](http://nation.marketo.com/community/support_solutions)。
