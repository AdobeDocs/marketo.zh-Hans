# 最佳下注 {#best-bets}

“最佳下注”选项卡包含一个列表，其中列出了所有基于优先级的热门潜在客户，该列表使用紧急度和相对分数进行计算。

通过单击“操作”列下的圆点菜单，您可以使用参与选项，例如：
* 发送Marketo电子邮件
* 添加到Marketo Campaign

您还可以从“最佳选择”选项卡中选择多个潜在客户，然后选择 _发送Marketo电子邮件_ 或 _添加到Marketo Campaign_.
