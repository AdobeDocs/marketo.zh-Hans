# 网络活动 {#web-activities}

Web活动选项卡显示潜在客户/联系人的Web活动。
查看潜在客户的最新Web活动，列出页面访问计数和相应的帐户。 您可以筛选结果以限制为指定的页数。

![](assets/web-activities-1.png)

## 匿名Web活动 {#anonymous-web-activities}

“匿名Web活动”选项卡显示所有&#x200B;**匿名**&#x200B;网页访客的Web活动。 查看引用页面访问计数的最新网站活动。
