# Web活动 {#web-activities}

“Web活动”选项卡显示潜在客户/联系人的Web活动。
查看您的潜在客户的最新Web活动，包括页面访问计数和相应的帐户。 您可以过滤结果以限制为指定的页面数。

# 匿名Web活动 {#anonymous-web-activities}

“匿名Web活动”选项卡显示所有&#x200B;**anonymous**&#x200B;网页访客Web活动。 查看引用页面访问计数的最新Web活动。
