# 如何在我的允许列表服务器上Sales Insight操作？ {#how-can-i-allowlist-sales-insight-actions-on-my-server}

遗憾的是，无法提供一个静态IP地址。 Sales Insight Actions已部署到云中，并利用200多台服务器处理电子邮件和用户请求。 提供一个非常广泛的范围是不安全的。
