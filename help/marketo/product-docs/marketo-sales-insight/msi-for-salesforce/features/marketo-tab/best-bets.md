# 最佳赌注{#best-bets}

“最佳下注”选项卡包括根据您的所有热门潜在客户的优先级列表所有热门潜在客户，这些潜在客户使用紧急程度和相对得分进行计算。

通过单击“操作”列下的“数据”菜单，您可以使用参与选项，例如：

* 发送Marketo电子邮件
* 添加到Marketo活动
* 视图 in Marketo
* 添加到观察列表

您还可以从“最佳下注”选项卡中选择多个潜在客户，然后选择&#x200B;_发送Marketo Email_&#x200B;或&#x200B;_添加到Marketo 活动_。
