# 洞察仪表板功能概述{#insights-dashboard-feature-overview}

了解有关销售洞察仪表板中可用功能的更多信息。

>[!PREREQUISITES]
>
>您必须拥有最新的MSI SFDC包和[configuration](/help/marketo/product-docs/marketo-sales-insight/msi-for-salesforce/features/configuration-for-existing-customers.md)。

![](assets/one.png)

## 联系布局{#contact-layout}

**啮合速度网格**

* 此智能网格包括过去90天里的有趣时刻、电子邮件和Web活动
* 用户可以选择“显示帐户”活动，这将在联系视图中分层所有帐户级别的有趣时刻、电子邮件和Web活动
* 用户可以突出显示特定周，以查看该周内的活动
* 默认视图:已选择当前周

**参与向下钻取和摘要**

* 向下钻取活动卡，以找到有趣的时刻、电子邮件和Web活动
* 有趣的瞬间活动卡 — 包括订阅选项
* 电子邮件活动卡 — 包含预览选项
* Web活动卡 — 包括单击链接的功能
* 每周摘要栏将显示该周的有趣时刻、电子邮件和Web活动。 每个图标都可单击，并可用作显示特定活动的过滤器
* 默认视图:这是当前列表的活动

**即将发布的电子邮件活动和事件**

“电子邮件活动”选项卡：

* 包括属于电子邮件项目或未来90天内计划的默认项目的活动
* 并非特定于联系人/潜在客户(即，活动列表是在Marketo实例中计划的所有电子邮件活动的通用列表)。 所有潜在客户、联系人、帐户和机会面板中的活动列表将相同
* 如果活动重复性使其在接下来的90天内运行三次以上，则在任何给定时间点（与Marketo中的行为类似）只会显示接下来的三次运行
* 此部分中的活动详细信息卡将包含预览选项。 如果流程包括多个发送电子邮件步骤，则所有电子邮件都将可供预览。 在发送电子邮件流步骤中，如果有多个“电子邮件选择”，则默认选项将可用于预览
* 过滤器包括“搜索”和“日期范围”

![](assets/three.png)

事件选项卡：

* 包括计划在未来90天内进行的事件项目
* 使用过滤器选项可查看所有事件/受邀事件（基于“管理员”设置）
* 选择已邀请的事件将显示已邀请特定联系人的事件以及成员状态
* 选择所有事件将显示未来90天内计划的事件的列表
* 活动详细信息卡在此部分中包含预览选项
* 筛选器包括“搜索”、“仅显示已邀请的事件”和“日期范围”

![](assets/two.png)
