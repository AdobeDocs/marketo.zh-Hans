# 洞察仪表板功能概述 {#insights-dashboard-feature-overview}

进一步了解销售洞察仪表板中的可用功能。

>[!NOTE]
>
>**先决条件**
>
>您必须拥有最新的MSI SFDC包和 [配置](http://docs.marketo.com/x/F4GMAg)。

![](assets/one.png)

## 联系人布局 {#contact-layout}

**啮合速度网格**

* 此智能网格包括过去90天内的有趣时刻、电子邮件和Web活动
* 用户可以选择“显示帐户”活动，这将在联系视图中分层所有帐户级别的有趣时刻、电子邮件和Web活动
* 用户可以突出显示特定周，以查看该周内的活动
* 默认视图:已选择当前周

**参与追溯和摘要**

* 深入查看活动卡，找到有趣的时刻、电子邮件和Web活动
* 有趣的时刻活动卡——包括订阅选项
* 电子邮件活动卡——包括预览选项
* Web活动卡——包括单击链接的功能
* 每周摘要栏将显示该周的有趣时刻、电子邮件和Web活动。 每个图标都可单击，并可用作过滤器以显示特定活动
* 默认视图:这是一列表活动的当前视图

**即将到来的电子邮件活动和事件**

电子邮件活动选项卡：

* 包括属于未来90天内计划的电子邮件项目或默认项目的活动
* 非特定于联系人／潜在客户(即，活动列表是在Marketo实例中计划的所有电子邮件活动的通用列表)。 所有潜在客户、联系人、帐户和机会面板中的活动列表将相同
* 如果活动重复出现，使其在接下来的90天内运行超过三次，则在任何给定时间点（与Marketo中的行为相似）只显示接下来的三次运行
* 此部分中的活动详细信息卡将具有预览选项。 如果流程包括多个发送电子邮件步骤，则所有电子邮件都可供预览。 在发送电子邮件流程步骤中，如果有多个“电子邮件选择”，则默认选项将可用于预览
* 过滤器包括“搜索”和“日期范围”

![](assets/three.png)

事件选项卡：

* 包括计划在未来90天内事件项目
* 使用筛选器选项查看所有事件/受邀事件（基于“管理员”设置）
* 选择已邀请的事件将显示已邀请特定联系人的事件以及成员状态
* 选择所有事件将显示未来90天内计划的事件的列表
* 活动详细信息卡在此部分中具有预览选项
* 筛选器包括“搜索”、“仅显示受邀事件”和“日期范围”

![](assets/two.png)

## 潜在客户布局 {#lead-layout}

**啮合速度网格**

* 此智能网格包括过去90天内的有趣时刻、电子邮件和Web活动
* 用户可以突出显示特定周，以查看该周内的活动

* “显示帐户活动”功能对潜在客户不可用，因为在转换为联系人之前，它不是Salesforce中任何帐户的一部分

**参与追溯和摘要**

* 展示有趣时刻、电子邮件和Web活动的深入活动卡
* 有趣的时刻活动卡——包括订阅选项
* 电子邮件活动卡——包括预览选项

* 每周摘要栏将显示该周的有趣时刻、电子邮件和Web活动。 每个图标都可单击，并可用作过滤器以显示特定活动
* 默认视图:这是一列表活动的当前视图

**即将到来的电子邮件活动和事件:**

电子邮件活动选项卡：

* 包括属于未来90天内计划的电子邮件项目或默认项目的活动
* 非特定于联系人／潜在客户(即，活动列表是在Marketo实例中计划的所有电子邮件活动的通用列表)。 所有潜在客户、联系人、帐户和机会面板中的活动列表将相同
* 如果活动重复出现，使其在接下来的90天内运行超过三次，则在任何给定时间点（与Marketo中的行为相似）只显示接下来的三次运行
* 此部分中的活动详细信息卡将具有预览选项。 如果流程包括多个发送电子邮件步骤，则所有电子邮件都可供预览。 在发送电子邮件流程步骤中，如果有多个“电子邮件选择”，则默认选项将可用于预览
* 过滤器包括“搜索”和“日期范围”

![](assets/five.png)

事件选项卡：

* 包括计划在未来90天内事件项目
* 使用筛选器选项查看所有事件/受邀事件（基于“管理员”设置）
* 选择已邀请的事件将显示已邀请特定联系人的事件以及成员状态
* 选择所有事件将显示未来90天内计划的事件的列表
* 活动详细信息卡在此部分中具有预览选项
