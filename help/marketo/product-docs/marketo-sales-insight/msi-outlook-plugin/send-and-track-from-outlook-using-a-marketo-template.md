# 使用Marketo模板从Outlook发送和跟踪 {#send-and-track-from-outlook-using-a-marketo-template}

如果您的营销团队向您提供了模板，下面介绍了在撰写电子邮件时如何使用这些模板来节省时间。

1. 打开Microsoft Outlook，然后单击 **Marketo消息**.

   ![](assets/image2014-9-23-17-3a8-3a33.png)

1. 选择所需的模板，进行预览，然后单击 **确定**.

   ![](assets/image2014-9-23-17-3a8-3a45.png)

1. 进行所有编辑，然后单击 **发送和跟踪**.

   ![](assets/image2014-9-23-17-3a8-3a58.png)

   >[!NOTE]
   >
   >加载项不支持令牌。 删除模板中可能包含的任何内容。

1. 查看预览，确保其显示良好，然后单击 **发送**.

   ![](assets/image2014-9-23-17-3a9-3a11.png)

   就这样！ 您能够利用您超棒的营销团队为您制作的模板节省大量时间。
