# 为基于IP的API访问创建允许列表 {#create-an-allowlist-for-ip-based-api-access}

有时，您只想授予API访问特定IP地址或某个地址范围的权限。 为此，首先要启用限制，然后指定允许使用API的IP地址。

>[!NOTE]
>
>**需要管理员权限**

1. 转到&#x200B;**[!UICONTROL 管理员]**&#x200B;区域。

   ![](assets/create-an-allowlist-for-ip-based-api-access-1.png)

1. 单击&#x200B;**[!UICONTROL Web服务]**。

   ![](assets/create-an-allowlist-for-ip-based-api-access-2.png)

1. 在&#x200B;**[!UICONTROL IP限制]**&#x200B;区域中，单击&#x200B;**[!UICONTROL 编辑]，**&#x200B;或单击左上角的&#x200B;**[!UICONTROL 编辑IP限制]**。

   ![](assets/create-an-allowlist-for-ip-based-api-access-3.png)

1. 列入允许列表选中&#x200B;**[!UICONTROL 启用IP限制]**&#x200B;框并输入要的IP地址。

   ![](assets/create-an-allowlist-for-ip-based-api-access-4.png)

   >[!NOTE]
   >
   >您可以输入单个IP地址或一个IP地址范围，也可以使用通配符。

1. 单击&#x200B;**[!UICONTROL 添加]**&#x200B;可打开其他字段以输入更多IP地址。

   ![](assets/create-an-allowlist-for-ip-based-api-access-5.png)

1. 单击&#x200B;**[!UICONTROL 保存]**。

   ![](assets/create-an-allowlist-for-ip-based-api-access-6.png)
