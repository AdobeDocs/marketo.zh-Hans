# 启用或禁用金库功能{#enable-or-disable-treasure-chest-features}

Treasure Chest提供了未完全支持的有趣实验功能。

>[!NOTE]
>
>**需要管理权限**

1. 在&#x200B;**Admin**&#x200B;下，单击&#x200B;**Treasure Chet**。

   ![](assets/image2014-9-16-17-3a0-3a36.png)

1. 单击&#x200B;**编辑**&#x200B;以获取要启用或禁用的功能。

   ![](assets/image2014-9-16-16-3a53-3a42.png)

1. 选中&#x200B;**已启用**&#x200B;框以启用，或取消选中禁用，然后单击&#x200B;**保存**。

   ![](assets/image2014-9-16-16-3a53-3a53.png)

   >[!TIP]
   >
   >您可能需要先注销，然后再重新登录Marketo，以查看更改生效。
