# 设置回退页面 {#set-a-fallback-page}

如果您的登陆页面处于离线状态或未找到，回退页面是最后一道防线。 确保你有一个。

>[!NOTE]
>
>**需要管理员权限**

1. 在 **管理员**，单击 **登陆页面**.

   ![](assets/image2014-9-10-12-3a7-3a22.png)

1. 在 **登陆页面** ，单击 **编辑**.

   ![](assets/image2014-9-10-12-3a7-3a5.png)

1. 输入 **回退页面** ，然后单击 **保存**.

   ![](assets/image2014-9-10-12-3a6-3a2.png)

瞧！ 您刚刚添加了一个备用页面！ 404页以上。
