# 设置回退页 {#set-a-fallback-page}

如果您的登陆页处于脱机状态或未找到，则回退页面是最后一道防线。 确保你有。

>[!NOTE]
>
>**需要管理员权限**

1. 在“管 **理员**”下，单击 **登陆页**。

   ![](assets/image2014-9-10-12-3a7-3a22.png)

1. 在登陆页 **选项卡** 下，单击 **编辑**。

   ![](assets/image2014-9-10-12-3a7-3a5.png)

1. 在对话 **框中输入** “回退”页面，然后单击 **保存**。

   ![](assets/image2014-9-10-12-3a6-3a2.png)

好！ 您刚刚添加了回退页面！ 404页。
