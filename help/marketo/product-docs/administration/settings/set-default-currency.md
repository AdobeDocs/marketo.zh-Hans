# 设置默认货币 {#set-default-currency}

了解如何查看和编辑Marketo Engage订阅的默认货币。

>[!NOTE]
>
>**需要管理员权限。**

1. 进入 **[!UICONTROL Admin]** 区域。

   ![](assets/set-default-currency-1.png)

1. 单击 **[!UICONTROL Location]**。

   ![](assets/set-default-currency-2.png)

1. 在&#x200B;**[!UICONTROL Edit]**&#x200B;中单击[!UICONTROL Subscription Currency Settings]。

   ![](assets/set-default-currency-3.png)

1. 选择您选择的货币格式并单击&#x200B;**[!UICONTROL Save]**。

   ![](assets/set-default-currency-4.png)
