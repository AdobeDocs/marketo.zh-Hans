# 使用 Adobe ID 登录用户 {#user-sign-in-with-adobe-id}

当具有Adobe标识的用户需要登录到Marketo Engage应用程序时，他们必须通过AdobeID登录链接而不是通过Marketo Engage登录页面上的典型登录进行登录。 单击该链接后，用户将被定向到Marketo Engage应用程序。

1. 在Marketo Engage登录页面上单击&#x200B;**[!UICONTROL Continue with AdobeID]**。

   ![](assets/user-sign-in-with-adobe-id-1.png)

1. 输入您的Adobe凭据并单击&#x200B;**[!UICONTROL Continue]**。

   ![](assets/user-sign-in-with-adobe-id-2.png)

成功登录后，您将被引导至Marketo Engage应用程序。
