# 编辑用户工作区 {#edit-user-workspaces}

1. 进入 **[!UICONTROL Admin]** 区域。

   ![](assets/edit-user-workspaces-1.png)

1. 单击 **[!UICONTROL Users & Roles]**。

   ![](assets/edit-user-workspaces-2.png)

1. 选择所需的用户并单击&#x200B;**[!UICONTROL Edit User]**。

   ![](assets/edit-user-workspaces-3.png)

1. 进行所需的更改并单击&#x200B;**[!UICONTROL Save]**。

   ![](assets/edit-user-workspaces-4.png)
