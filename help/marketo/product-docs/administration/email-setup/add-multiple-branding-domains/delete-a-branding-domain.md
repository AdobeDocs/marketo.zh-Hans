# 删除品牌域 {#delete-a-branding-domain}

1. 转到 **管理员** 的上界。

   ![](assets/delete-a-branding-domain-1.png)

1. 单击 **电子邮件**.

   ![](assets/delete-a-branding-domain-2.png)

1. 在“品牌化域名”(Branding Domains)表格中，选择要删除的域，然后单击 **删除**.

   ![](assets/delete-a-branding-domain-3.png)

   >[!NOTE]
   >
   >如果要删除主品牌域，必须首先选择其他品牌域作为主品牌域。
