# 支持的浏览器 {#supported-browsers}

Marketo当前支持以下浏览器：

| 浏览器 | 支持的版本 |
|---|---|
| [Google Chrome](http://www.google.com/intl/en/chrome/browser/) | 最新 |
| [Mozilla Firefox](http://www.mozilla.org/en-US/firefox/new/) | 最新 |
| [Apple Safari](http://support.apple.com/downloads/#safari) | 最新 |
| [Microsoft Edge](https://www.microsoft.com/en-us/windows/microsoft-edge) | 最新 |

为获得最佳体验，请 **保持浏览器最新**!
