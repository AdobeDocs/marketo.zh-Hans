# 蒙奇金 {#munchkin}

Munchkin是您在网站上放置的JavaScript代码片段。 它可跟踪访问次数、点击次数和表单填写。 它还有一个丰富的API，可以做各种很酷的事情。

查看 [Munchkin上的开发人员页面](https://developers.marketo.com/documentation/websites/lead-tracking-munchkin-js/){target=&quot;_blank&quot;}。
