# 命名帐户概述 {#named-account-overview}

指定帐户会接收您所定位公司的人员。 功能板为所有指定帐户提供每个属性的当前数据。

## 指定帐户功能板 {#named-accounts-dashboard}

![](assets/one.png)

>[!TIP]
>
>默认情况下，指定帐户按创建日期排序，但您可以按标题中带有排序图标的任何列进行排序。

>[!NOTE]
>
>Marketo会将同步到Marketo的所有CRM用户显示为“帐户所有者”或“帐户团队成员”的筛选器值。
