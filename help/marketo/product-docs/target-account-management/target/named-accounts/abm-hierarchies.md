# ABM层次{#abm-hierarchies}

层次结构使ABM用户能够继承其CRM中指定帐户之间的父/子关系。

## ABM等级是什么？{#what-are-abm-hierarchies}

公司可以有多个部门和子公司。 这些公司通常通过称为层次的父子关系来组织自己。 ABM可以从您的SFDC或MSD集成继承这些层次结构，并允许您作为单个系列使用不同的部门。

## 使用ABM层次{#working-with-abm-hierarchies}

通过ABM层次结构，您可以在指定帐户仪表板中快速获取有关整个层次结构或单个帐户的信息。
