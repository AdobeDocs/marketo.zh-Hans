# 创建[!UICONTROL Named Account] {#create-a-named-account}

按照以下步骤手动创建指定帐户。

1. 在[!UICONTROL Named Accounts]中，单击&#x200B;**[!UICONTROL New]**&#x200B;下拉菜单并选择&#x200B;**[!UICONTROL Create Named Account]**。

   ![](assets/two-1.png)

1. 填写所需字段并单击&#x200B;**[!UICONTROL Create]**。

   ![](assets/three-1.png)

   >[!TIP]
   >
   >直接单击指定帐户可查看其仪表板。
