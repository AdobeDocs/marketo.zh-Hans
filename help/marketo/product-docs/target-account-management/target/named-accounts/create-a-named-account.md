# 创建指定帐户 {#create-a-named-account}

按照以下步骤手动创建指定帐户。

1. 在指定帐户中，单击 **新** 下拉菜单并选择 **创建指定帐户**.

   ![](assets/two-1.png)

1. 填写所需的字段并单击 **创建**.

   ![](assets/three-1.png)

   >[!TIP]
   >
   >直接单击指定帐户可查看其仪表板。
