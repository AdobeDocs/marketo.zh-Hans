# 向帐户列表{#add-an-existing-named-account-to-an-account-list}添加现有指定帐户

将指定帐户添加到帐户列表很简单。

>[!NOTE]
>
>这仅适用于帐户列表,**不适用于**&#x200B;动态帐户列表。

1. 选择要添加到的已命名帐户的行。

   ![](assets/four-1.png)

1. 单击&#x200B;**命名帐户操作**&#x200B;下拉框，然后选择&#x200B;**添加到帐户列表**。

   ![](assets/five-1.png)

1. 单击&#x200B;**帐户列表**&#x200B;下拉列表，选择所需的帐户列表，然后单击&#x200B;**添加**。

   ![](assets/six-1.png)

   就这样！
