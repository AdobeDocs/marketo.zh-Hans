# 帐户列表分析{#account-list-insights}

“帐户列表”仪表板优惠视图该列表内所有指定帐户的汇总洞察。

>[!NOTE]
>
>Marketo TAM会根据生成的渠道或帐户参与分数自动确定帐户列表中最热门的已命名帐户。

## 帐户列表仪表板{#account-list-dashboard}

要查看帐户列表的仪表板，只需单击其名称……

![](assets/one-new.png)

...然后仪表板出现。

![](assets/two-new-1.png)

## “指定帐户”选项卡{#named-accounts-tab}

单击&#x200B;**命名帐户**&#x200B;选项卡，查看哪些已命名帐户属于该帐户列表。

![](assets/three-1.png)

>[!NOTE]
>
>您可以通过选择此选项卡并单击&#x200B;**删除指定帐户**&#x200B;来删除该指定帐户。

## 选项卡{#used-by-tab}

单击&#x200B;**使用者**&#x200B;选项卡，查看哪些资产引用了该帐户列表。

![](assets/four-2.png)
