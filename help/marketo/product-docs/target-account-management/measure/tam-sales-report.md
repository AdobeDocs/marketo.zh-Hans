# TAM销售报告{#tam-sales-report}

每周收到一封电子邮件，其中包含您的热门帐户及其每周趋势。

>[!NOTE]
>
>了解如何在[此处](/help/marketo/product-docs/target-account-management/measure/tam-report-setup.md)设置此报表。

报告分享：

* 按您选择的帐户分数排序的已命名帐户
* 最热衷的人
* 关键趋势和有趣时刻
* 用于下载包含其他详细信息的CSV文件的链接

![](assets/tam-sales-report-1.png)
