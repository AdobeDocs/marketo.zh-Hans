# TAM销售报表 {#tam-sales-report}

每周收到一封电子邮件，其中包含您的热门帐户及其每周趋势。

>[!NOTE]
>
>了解如何设置此报表 [此处](/help/marketo/product-docs/target-account-management/measure/tam-report-setup.md).

报告将分享：

* 按您选择的帐户得分排序的指定帐户
* 主要参与人员
* 主要趋势和有趣的时刻
* 用于下载包含其他详细信息的CSV文件的链接

![](assets/tam-sales-report-1.png)
