# 查找上传的图像或文件的URL {#find-the-url-of-an-uploaded-image-or-file}

是否正在查找您上传的图像或文件的网址(URL)? Marketo掩护你。

1. 转到 **Design Studio**.

   ![](assets/designstudio-4.png)

1. 单击 **图像和文件** 在左侧菜单中，然后选择您想要的URL文件。

   ![](assets/image2014-9-25-14-3a47-3a53.png)

1. 的 **URL** 显示在详细信息页面上。 右键单击 **URL** 单击 **复制** 中。

   ![](assets/image2014-9-25-14-3a48-3a16.png)

给自己拍个背，找份干得不错的工作！
