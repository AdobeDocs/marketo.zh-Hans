# 创建对话 {#create-a-dialogue}

下面是如何创建新对话框。

1. 单击 **[!UICONTROL Dialogues]**。

   ![](assets/create-a-dialogue-1.png)

1. 单击&#x200B;**[!UICONTROL Create New]**&#x200B;按钮。

   ![](assets/create-a-dialogue-2.png)

1. 选择一个空白对话框，或其中一个预填充模板。 输入名称（说明是可选的），更改优先级（可选），然后单击&#x200B;**[!UICONTROL Create]**。

   ![](assets/create-a-dialogue-3.png)

>[!NOTE]
>
>优先级确定当访客符合同时使用多个对话框的资格时，将向访客显示的对话框。
