# 活动日志 {#activity-log}

查看所有任务的列表及其随附的详细信息，包括名称、所有者、类型以及编辑这些任务的人和时间。

1. 在“生成AI”下，单击&#x200B;**[!UICONTROL Assisted responses]**。

   ![](assets/activity-log-1.png)

1. 单击&#x200B;**[!UICONTROL Activity log]**&#x200B;选项卡。

   ![](assets/activity-log-2.png)

1. 实例的所有活动都将显示在一个位置。
