# 使用限制 {#usage-limits}

请访问“使用限制”页面以查看重要信息，例如包信息和使用限制状态。

1. 在配置下，单击&#x200B;**[!UICONTROL Usage limits]**。

   ![](assets/usage-limits-1.png)

1. 查看当前包、参与对话的每月限制状态以及日历/实时聊天名额的生命周期限制状态。

   ![](assets/usage-limits-2.png)

>[!NOTE]
>
>每月限制会在每月1日自动重置。
