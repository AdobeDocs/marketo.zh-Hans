# 聊天机器人行为 {#chatbot-behavior}

下面是访客在每个聊天机器人预期行为之外的不同可能情景。

<table>
  <tbody>
    <tr>
      <th>缩写</th>
      <th>详细信息</th>
    </tr>
    <tr>
      <td>D1、D2、D3等</td>
      <td>表示多个对话框，其中D1为对话框1</td>
    </tr>
    <tr>
      <td>P1、P2、P3等</td>
      <td>表示P1为最高优先级的对话优先级</td>
    </tr>
    <tr>
      <td>WP1、WP2、WP3等</td>
      <td>表示多个网页，其中WP1是第一个网页</td>
    </tr>
    <tr>
      <td>V1、V2、V3等</td>
      <td>表示多个网页访客，其中V1为访客1</td>
    </tr>
   </tbody>
</table>
