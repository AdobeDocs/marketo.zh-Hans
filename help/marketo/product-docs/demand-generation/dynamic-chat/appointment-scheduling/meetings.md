# 会议 {#meetings}

在这里，您可以看到网站访客通过各种对话框安排的所有约会。 您可以在此处找到预订约会的人员的电子邮件地址、他们预订约会的代理、预定约会的时间，以及预定会议时间是否已过。

![](assets/meetings-1.png)

>[!NOTE]
>
>在座席日历上预订会议时，座席将收到一封有关预订的电子邮件通知，其中包括有关访客Dynamic Chat参与的详细信息。
