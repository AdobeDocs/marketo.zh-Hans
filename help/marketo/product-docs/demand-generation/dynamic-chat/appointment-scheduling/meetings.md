# 会议 {#meetings}

在这里，您将看到网站访客通过各种对话安排的所有约会。 在此，您将找到预订约会的人的电子邮件地址、他们与哪位座席预订了约会、预定约会何时发生以及计划会议时间是否已过。

![](assets/meetings-1.png)

>[!NOTE]
>
>在座席日历上预订会议后，座席将收到有关预订的电子邮件通知，包括有关访客动态聊天参与的详细信息。
