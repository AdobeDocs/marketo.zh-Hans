# 创建交互式网络研讨会 {#create-an-interactive-webinar}

简介。

1. 转到 **营销活动**.

   ![](assets/create-an-interactive-webinar-1.png)

1. 右键单击所需的文件夹并选择 **新计划**.

PICC

1. 命名您的事件。 在“程序类型”(Program Type)下，选择 **事件**.

PICC

1. 单击渠道下拉列表，然后选择 **网络研讨会**.

PICC

1. 选择 **交互式网络研讨会** 单击 **创建**.

PICC

>
