# 配置事件设置并将Marketo与您的网络研讨会同步 {#configure-event-settings-and-sync-marketo-with-your-webinar}

按照以下步骤配置Marketo事件设置并连接Marketo和ON24。

## 设置事件 {#set-the-event}

1. 选择要与ON24网络研讨会关联的事件，然后单击&#x200B;**[!UICONTROL Event Actions]**&#x200B;下拉框并选择&#x200B;**[!UICONTROL Event Settings]**。

   ![](assets/one.png)

1. 选择ON24作为[!UICONTROL Event Partner]。

   ![](assets/two.png)

1. 选择[!UICONTROL Login]帐户（例如，显示名称）。

   ![](assets/three.png)

1. 输入[!UICONTROL Event Id] （从ON24获取）。 单击 **[!UICONTROL Save]**。

   ![](assets/four.png)

   >[!NOTE]
   >
   >在高峰时段，ON24可能需要15到20分钟才能将事件信息提供给Marketo。 如果您收到“会话ID无效”消息，请稍后重试。

## 设置计划 {#set-the-schedule}

设置与ON24网络研讨会关联的事件时，事件计划会使用来自ON24的数据进行填充。 要访问[!UICONTROL Event Schedule]对话框，请执行以下步骤。

1. 选择事件。 单击&#x200B;**[!UICONTROL Event Actions]**&#x200B;下拉菜单并选择&#x200B;**[!UICONTROL Schedule].**

   ![](assets/five.png)

1. 选择您的&#x200B;**[!UICONTROL Start Date]**、**[!UICONTROL End Date]**&#x200B;和&#x200B;**[!UICONTROL Time Zone]**。 单击 **[!UICONTROL Save]**。

   ![](assets/six-1.png)

   >[!NOTE]
   >
   >如果您在ON24中更新任何事件信息，则必须在&#x200B;**[!UICONTROL Refresh from Webinar Provider]**&#x200B;菜单中单击[!UICONTROL Event Actions]以查看新数据填充。

现在您可以继续下一步了：[创建子营销活动和本地资源](/help/marketo/product-docs/demand-generation/events/create-an-event/create-an-event-with-the-marketo-on24-adapter/create-child-campaigns-and-local-assets.md){target="_blank"}。
