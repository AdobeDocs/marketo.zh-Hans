# 在ON24中创建网络研讨会活动 {#create-your-webinar-event-in-on}

>[!IMPORTANT]
>
>自2022年8月起，ON24不再支持新的Marketo集成。 本文中的信息仅适用于现有用户。

在ON24中创建网络研讨会后，您可以创建一个Marketo活动并将其与ON24网络研讨会相关联。 这允许系统共享注册和考勤信息。 记下URL和其他信息，以供在确认电子邮件和ICS文件中使用。

在创建ON24网络研讨会时，有些设置由Marketo使用，而其他设置则仅由ON24使用。 Marketo使用以下设置：

**标题和描述**

* 事件标题 — 网络研讨会的名称。 该内容可在Marketo中查看。
* 事件摘要（可选） — 网络研讨会的描述。 该内容可在Marketo中查看。

**日期和时间**

* 实时事件开始 — 开始日期和时间。 该内容可在Marketo中查看。
* 实时结束时间 — 完成日期和时间。 该内容可在Marketo中查看。

下一步是 [在Marketo中创建事件](/help/marketo/product-docs/demand-generation/events/create-an-event/create-an-event-with-the-marketo-on24-adapter/create-an-event-in-marketo.md).
