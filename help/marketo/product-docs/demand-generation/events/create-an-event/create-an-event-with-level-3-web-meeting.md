# 创建具有第3级Web会议的事件 {#create-an-event-with-level-web-meeting}

>[!PREREQUISITES]
>
>* [将第3级Web会议添加为LaunchPoint服务](/help/marketo/product-docs/administration/additional-integrations/add-level-3-web-meeting-as-a-launchpoint-service.md)
>* [创建新事件程序](/help/marketo/product-docs/demand-generation/events/understanding-events/create-a-new-event-program.md)
>* 设置相应的 [流量操作](/help/marketo/product-docs/core-marketo-concepts/smart-campaigns/flow-actions/add-a-flow-step-to-a-smart-campaign.md)跟踪参与度


首先在级别3中创建网络研讨会。 如果您需要帮助，请查看 [第3级资源库](https://www.level3.com/en/resource-library/). 你会发现它和BrightTalk非常相似。  Marketo使用少量级别3字段的子集：

* **名称**  — 网络广播的名称。
* **开始日期** 网络广播的开始日期。
* **结束日期**  — 网络广播的结束日期。
* **时区**  — 为网播设置的时区。
* **描述**  — 网播描述。

1. 选择新事件。 单击 **事件操作、** then **事件设置。**

   ![](assets/image2016-3-24-15-3a40-3a39.png)

1. 在事件合作伙伴下，选择 **第3级Web会议**.

   ![](assets/image2016-3-24-15-3a42-3a10.png)

1. 在“登录”下，选择您的Level 3登录。

   ![](assets/image2016-3-24-15-3a43-3a43.png)

1. 在“事件”下，选择要使用的级别3事件。

   ![](assets/image2016-3-24-15-3a44-3a41.png)

1. 单击 **保存**.

   ![](assets/image2016-3-24-15-3a45-3a31.png)

   太棒了！ 您已将活动连接到级别3!

## 查看计划  {#viewing-the-schedule}

在项目计划视图中，单击事件的日历条目。 您可以在屏幕右侧看到计划！
