# 了解广告网络集成 {#understanding-ad-network-integrations}

![](assets/hith-golden-gate-144833144-e.jpeg)

## 广告网络集成的工作原理 {#how-ad-network-integrations-work}

广告网络集成将Marketo的独特受众数据连接到您的付费媒体系统，以便您能够运行更具针对性、个性化的数字广告营销活动。 这些集成当前可与Facebook、LinkedIn和Google配合使用。

功能包括：

* 向付费媒体系统发送受众，以便通过个性化展示广告来定位已知的潜在客户，以培养或重新激活他们

   * 按购买者阶段、参与级别等进行分段
   * 扩展受众以改进客户获取
   * 使用广告定位电子邮件订阅者和未响应者
   * 跨渠道培养，例如电子邮件和数字广告

* 将潜在客户或数据从付费媒体系统摄取到Marketo

   * 例如，Facebook用户单击“订阅”按钮并提供名称和电子邮件地址

* 将离线转化数据上传到付费媒体系统，优化潜在客户获取并培养员工精力

   * 例如，人员搜索、查看并单击广告，然后在线填写表单。 之后，他将离线收到演示，并成为CRM机会。 广告商就是在那里瞄准他

* 使用 [Web个性化](/help/marketo/product-docs/web-personalization/understanding-web-personalization/web-personalization-overview.md)，以将其驱回您的网站

   * 按区段或垂直进行个性化
