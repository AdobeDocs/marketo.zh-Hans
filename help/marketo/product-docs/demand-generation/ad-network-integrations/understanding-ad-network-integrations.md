# 了解广告网络集成{#understanding-ad-network-integrations}

## 广告网络集成如何工作{#how-ad-network-integrations-work}

广告网络集成将Marketo的独特受众数据与您的付费媒体系统相连，使您能够运行更具针对性、个性化的数字广告活动。 这些集成当前可与Facebook、LinkedIn和Google一起使用。

功能包括：

* 将受众发送到付费媒体系统，以便用个性化的展示广告目标已知潜在客户，以培养或重新激活它们

   * 按买家阶段、参与程度等进行细分
   * 扩展受众以改进客户赢取
   * 目标电子邮件订阅者和无响应者（含广告）
   * 在渠道（例如电子邮件和数字广告）中进行培养

* 将潜在客户或数据从付费媒体系统引入Marketo

   * 例如，Facebook用户单击“订阅”按钮并提供名称和电子邮件地址

* 将线下转化数据上传到付费媒体系统，优化销售线索获取并培养员工

   * 例如，人员搜索、查看和单击广告，然后在线填写表单。 之后，他将脱机收到一个演示并成为CRM机会。 广告商就是在这目标他

* 使用[Web个性化](/help/marketo/product-docs/web-personalization/understanding-web-personalization/web-personalization-overview.md)重新定位网站访客，将其推回您的网站

   * 按细分或垂直进行个性化
